## 📋 使用指南

### 按年龄选择平台

| 年龄段 | 推荐平台 | 推荐内容 |
|--------|----------|----------|
| 6-8岁 | B站中文教程 | Scratch图形化编程 |
| 8-10岁 | B站 + YouTube | Micro:bit MakeCode |
| 10-12岁 | B站中文教程 | Arduino + 传感器 |
| 12-14岁 | YouTube英文 | Arduino机器人制作 |
| 14岁+ | YouTube + B站 | ESP32/MicroPython |

### 学习建议

1. **先看入门视频**，了解整体框架
2. **跟着教程动手做**，不要只看不练
3. **遇到问题看评论区**，通常有解决方案
4. **收藏播放列表**，系统学习更高效

## 💡 六、学习建议

### 视频学习技巧

1. **1.5倍速观看**：提高效率，重要内容减速
2. **边看边做**：暂停视频，动手实践
3. **记笔记**：记录关键步骤和问题
4. **看评论区**：常见问题通常有人问过
5. **收藏播放列表**：系统学习更高效

### 常见问题

| 问题 | 解决方案 |
|------|----------|
| 视频太长看不完 | 分多次观看，每次30分钟 |
| 跟不上教程节奏 | 使用0.75倍速，多暂停 |
| 英文听不懂 | 开启自动翻译字幕 |
| 缺少硬件 | 使用Tinkercad仿真 |

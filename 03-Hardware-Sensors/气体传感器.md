# 👃 气体传感器 (MQ Series Gas Sensor)

## 🎯 一句话定义
气体传感器就像机器人的鼻子，能闻到危险气体，保护我们的安全。

## 🧒 给孩子的解释
你的鼻子能闻到香喷喷的饭菜，也能闻到臭臭的味道。机器人的"鼻子"更厉害，它能闻到我们闻不到的危险气体！比如煤气泄漏、烟雾、酒精等。如果家里煤气泄漏了，它能马上报警，让全家人安全撤离。就像一个24小时值班的安全卫士！

## 🔬 工作原理

### 半导体气体传感器
```
核心材料：二氧化锡(SnO2)

清洁空气中：
  SnO2表面吸附氧气 → 形成势垒 → 电阻高

目标气体中：
  气体与吸附氧反应 → 释放电子 → 电阻降低

气体浓度越高 → 电阻越低 → 输出电压越高
```

## 📊 技术参数

| 参数 | 数值 | 说明 |
|------|------|------|
| 工作电压 | 5V DC | 必须5V供电 |
| 加热电压 | 5V DC | 加热丝需要 |
| 加热电流 | 约150mA | 耗电较大 |
| 检测方式 | 模拟+数字 | 双输出 |
| 预热时间 | 24-48小时 | 首次使用需要 |
| 响应时间 | <10秒 | 快速响应 |
| 恢复时间 | <30秒 | 恢复到清洁空气 |
| 工作温度 | -10°C到50°C | 室内使用 |
| 尺寸 | 直径约20mm | 中等大小 |

### 引脚说明
```
VCC  ──── 5V电源正极
GND  ──── 电源负极
DO   ──── 数字输出（超过阈值为HIGH）
AO   ──── 模拟输出（气体浓度）
```

### ⚠️ 重要警告

#### 预热时间
```
首次使用：必须预热24-48小时
日常使用：每次上电预热5分钟
预热期间：数据不准确，不要使用
```

#### 安全注意
```
1. 气体传感器不能替代专业检测设备
2. 检测可燃气体时要远离火源
3. 不要在高浓度气体中长时间使用
4. 传感器会老化，需要定期校准
```

### Arduino 接线示例
```
MQ传感器      Arduino
VCC      ──>  5V
GND      ──>  GND
AO       ──>  A0（模拟输出）
DO       ──>  D2（数字输出，可选）
```

### Arduino 代码示例
```cpp
#define SENSOR_PIN A0

void setup() {
  Serial.begin(9600);
  Serial.println("预热中，请等待5分钟...");
  delay(300000);  // 等待5分钟预热
}

void loop() {
  int gasValue = analogRead(SENSOR_PIN);
  
  Serial.print("气体浓度: ");
  Serial.println(gasValue);
  
  if (gasValue > 300) {
    Serial.println("警告：检测到气体！");
  }
  
  delay(1000);
}
```

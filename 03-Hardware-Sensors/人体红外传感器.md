# 👀 人体红外传感器 (PIR Motion Sensor HC-SR501)

## 🎯 一句话定义
人体红外传感器能看见热量！就像蛇一样的热感应超能力，能检测到人或动物的存在。

## 🧒 给孩子的解释
你知道蛇是怎么在黑暗中抓到老鼠的吗？因为蛇能"看见"热量！人体红外传感器就有类似的能力。人和动物的身体会发出一种看不见的"热光"（红外线），这个传感器就能检测到。当有人走过的时候，它就知道"有人来了！"这样你就可以做一个自动开灯的装置，或者一个防盗报警器！

## 🔬 工作原理

### 被动红外检测
```
人体（37°C）→ 发出红外线 → 传感器检测到 → 输出高电平

被动红外：传感器本身不发光，只接收
主动红外：传感器发光，检测反射（如红外避障）
```

### 内部结构
```
┌─────────────────────────┐
│      菲涅尔透镜          │  ← 聚焦红外线
│  ┌───────────────────┐  │
│  │    热释电传感器     │  │  ← 检测红外变化
│  └───────────────────┘  │
│  ┌───────────────────┐  │
│  │    信号处理电路     │  │  ← 放大、比较
│  └───────────────────┘  │
└─────────────────────────┘
```

### 工作流程
1. **红外聚焦**：菲涅尔透镜将红外线聚焦到传感器
2. **热释电效应**：温度变化产生电信号
3. **信号放大**：微弱信号被放大
4. **比较判断**：与阈值比较，判断是否有人
5. **输出**：检测到人输出高电平

### 菲涅尔透镜的作用
```
无透镜：检测范围小，灵敏度低
有透镜：检测范围大，灵敏度高
        形成多个检测区域（锥形）
```

## 📊 技术参数

| 参数 | 数值 | 说明 |
|------|------|------|
| 工作电压 | 4.5V - 20V | 宽电压范围 |
| 工作电流 | 约50μA | 很省电 |
| 检测距离 | 3 - 7米 | 可调节 |
| 检测角度 | 约100° | 锥形范围 |
| 输出方式 | 数字输出 | 高/低电平 |
| 延时时间 | 可调节（0.3-300秒） | 触发后持续时间 |
| 封锁时间 | 可调节（2.5秒） | 两次触发最小间隔 |
| 触发方式 | 可选择 | 不可重复触发/可重复触发 |
| 尺寸 | 直径约25mm | 小巧 |

### 引脚说明
```
VCC  ──── 4.5V-20V电源
OUT  ──── 数字输出（检测到人为HIGH）
GND  ──── 电源负极
```

### 调节旋钮
```
灵敏度调节（Sx）：
  顺时针 → 灵敏度增加，检测距离变远
  逆时针 → 灵敏度降低，检测距离变近

延时调节（Tx）：
  顺时针 → 延时变长（最长约300秒）
  逆时针 → 延时变短（最短约0.3秒）
```

### 触发模式选择
```
跳线帽选择：
  L = 不可重复触发：触发后输出高电平，延时结束后变低
  H = 可重复触发：持续检测到人就一直输出高电平
```

### Arduino 接线示例
```
HC-SR501     Arduino
VCC      ──>  5V
OUT      ──>  D2
GND      ──>  GND
```

### Arduino 代码示例
```cpp
#define PIR_PIN 2

void setup() {
  pinMode(PIR_PIN, INPUT);
  Serial.begin(9600);
  Serial.println("等待传感器稳定（约30秒）...");
  delay(30000);  // 等待传感器稳定
}

void loop() {
  int motion = digitalRead(PIR_PIN);
  
  if (motion == HIGH) {
    Serial.println("检测到人体移动！");
  } else {
    Serial.println("未检测到移动");
  }
  
  delay(500);
}
```

---

> 💡 **小贴士**：PIR传感器上电后需要约30秒的稳定时间，这段时间内可能会误触发。另外，传感器对快速移动的物体更灵敏，对静止或缓慢移动的物体检测效果差。安装时要注意高度（建议2-2.5米）和角度，避免正对窗户（阳光干扰）或空调（热风干扰）。

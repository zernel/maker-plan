# 🏎️ 红外循迹传感器 (IR Line Tracking Sensor)

## 🎯 一句话定义
红外循迹传感器就像机器人的眼睛，能分辨地面上的黑白线条，让机器人沿着线路行驶。

## 🧒 给孩子的解释
你见过赛车跑道上的白线吗？红外循迹传感器就像给机器人装了一双能看线的眼睛！它发出一种看不见的光（红外线），然后看这个光有没有被反射回来。白线会把光反射回来，黑线会把光吸收掉。这样机器人就知道自己是在白线上还是黑线上了，就能沿着跑道跑啦！

## 🔬 工作原理

### 红外发射与接收
```
传感器 ──发出红外光──> 地面
传感器 <──接收反射光── 地面

白色地面：反射强 → 传感器收到光 → 输出低电平
黑色地面：反射弱 → 传感器收不到光 → 输出高电平
```

### 核心原理
1. **发射**：红外LED发出红外光照射地面
2. **反射**：地面颜色不同，反射强度不同
3. **接收**：光电二极管接收反射光
4. **判断**：根据反射光强度判断是黑还是白

### 多传感器组合
```
典型配置：3-5个传感器一字排开

    [左2] [左1] [中间] [右1] [右2]
      ↓     ↓      ↓      ↓     ↓
    ───────────────────────────────
           黑色循迹线
```

## 📊 技术参数

| 参数 | 数值 | 说明 |
|------|------|------|
| 工作电压 | 3.3V - 5V | 兼容多种开发板 |
| 工作电流 | 约20mA | 每个传感器 |
| 检测距离 | 0 - 3cm | 离地高度 |
| 输出方式 | 数字输出 | 高/低电平 |
| 检测颜色 | 黑白为主 | 其他颜色效果不同 |
| 响应时间 | <1ms | 反应很快 |
| 指示灯 | LED | 检测到黑线亮灯 |
| 尺寸 | 约30×15mm | 小巧 |

### 引脚说明
```
VCC  ──── 3.3V或5V电源
GND  ──── 电源负极
DO   ──── 数字输出（检测到黑线为HIGH）
AO   ──── 模拟输出（可选，返回反射强度）
```

### 调节灵敏度
大多数模块有电位器可以调节：
- **顺时针**：增加灵敏度，检测距离变远
- **逆时针**：降低灵敏度，检测距离变近

## 🎮 项目创意

### 入门项目
- **单线循迹**：沿着一条黑线行驶
- **十字路口**：识别十字路口并转弯

### 进阶项目
- **复杂路线**：循迹+避障组合
- **PID循迹**：使用PID算法实现平滑行驶
- **多线选择**：识别并选择不同颜色的线路

### 创意项目
- **迷宫求解**：在迷宫中找到出口
- **仓库机器人**：模拟仓库搬运路线
- **相扑机器人**：在竞技场中保持在线内

## 🔌 兼容性

| 开发板 | 支持 | 说明 |
|--------|------|------|
| Micro:bit | ✅ | 使用P0-P2读取数字信号 |
| Arduino | ✅ | 最常用，直接读取数字引脚 |
| ESP32 | ✅ | 支持，注意逻辑电平 |
| Raspberry Pi | ✅ | 需要GPIO库 |

### Arduino 接线示例（5路循迹）
```
循迹模块      Arduino
VCC      ──>  5V
GND      ──>  GND
DO1      ──>  D2（最左）
DO2      ──>  D3
DO3      ──>  D4（中间）
DO4      ──>  D5
DO5      ──>  D6（最右）
```

### Micro:bit 接线示例（3路循迹）
```
循迹模块      Micro:bit
VCC      ──>  3V
GND      ──>  GND
DO1      ──>  P0（左）
DO2      ──>  P1（中）
DO3      ──>  P2（右）
```

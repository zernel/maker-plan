# 🤸 陀螺仪传感器 (MPU6050 IMU)

## 🎯 一句话定义
陀螺仪传感器就像内耳的平衡系统，能感知机器人的旋转和倾斜。

## 🧒 给孩子的解释
你闭着眼睛单脚站立，为什么不会摔倒？因为你耳朵里有一个"平衡系统"！陀螺仪传感器就是给机器人装的"平衡系统"。它能感觉到机器人在往哪个方向倾斜，转得有多快。有了它，机器人就能自己保持平衡，或者知道自己的手臂抬到了什么角度。就像玩体感游戏时，手柄知道你往哪边挥一样！

## 🔬 工作原理

### 六轴运动追踪
```
MPU6050 = 3轴加速度计 + 3轴陀螺仪

加速度计：测量线性加速度（包括重力）
         X轴：左右倾斜
         Y轴：前后倾斜
         Z轴：上下运动

陀螺仪：测量角速度（旋转速度）
       X轴：俯仰（Pitch）
       Y轴：翻滚（Roll）
       Y轴：偏航（Yaw）
```

### 坐标系说明
```
        Z轴（向上）
         ↑
         │
         │
         └────────→ Y轴（向右）
        /
       /
      ↙
    X轴（向前）
```

### 数据融合
```
加速度计：长期稳定，但有噪音
陀螺仪：短期精确，但会漂移
融合算法：互补滤波/卡尔曼滤波
         → 得到精确的姿态角
```

## 📊 技术参数

| 参数 | 数值 | 说明 |
|------|------|------|
| 工作电压 | 3.3V - 5V | 兼容多种开发板 |
| 通信接口 | I2C | SDA/SCL两线 |
| I2C地址 | 0x68 或 0x69 | AD0引脚决定 |
| 加速度量程 | ±2g/±4g/±8g/±16g | 可设置 |
| 陀螺仪量程 | ±250/±500/±1000/±2000°/s | 可设置 |
| ADC精度 | 16位 | 高精度 |
| 采样率 | 最高1kHz | 可设置 |
| 温度传感器 | 内置 | -40°C到85°C |
| 尺寸 | 约20×15mm | 小巧 |

### 引脚说明
```
VCC  ──── 3.3V或5V电源
GND  ──── 电源负极
SCL  ──── I2C时钟线
SDA  ──── I2C数据线
INT  ──── 中断输出（可选）
AD0  ──── I2C地址选择（0=0x68, 1=0x69）
```

## 🎮 项目创意

### 入门项目
- **倾斜检测器**：检测物体倾斜角度
- **姿态显示器**：在屏幕上显示姿态

### 进阶项目
- **自平衡机器人**：两轮自平衡小车
- **体感控制器**：用手势控制游戏
- **飞行控制器**：四轴飞行器的核心

### 创意项目
- **手势识别**：识别不同的手势动作
- **运动追踪**：追踪手臂运动轨迹
- **防倾倒报警**：物体倾斜超过阈值报警

```
MPU6050      Arduino

GND      ──>  GND
SCL      ──>  A5
SDA      ──>  A4
```

### Arduino 代码示例
```cpp
#include <Wire.h>
#include <MPU6050.h>

MPU6050 mpu;

void setup() {
  Serial.begin(9600);
  Wire.begin();
  mpu.initialize();
}

void loop() {
  int16_t ax, ay, az;
  int16_t gx, gy, gz;
  
  mpu.getMotion6(&ax, &ay, &az, &gx, &gy, &gz);
  
  Serial.print("加速度: ");
  Serial.print(ax); Serial.print(", ");
  Serial.print(ay); Serial.print(", ");
  Serial.println(az);
  
  delay(100);
}
```

> 💡 **小贴士**：MPU6050需要校准才能得到准确的数据。上电后要保持静止几秒钟，让传感器校准零点。另外，I2C总线需要上拉电阻（4.7kΩ），大多数模块已内置。如果多个I2C设备共用，要注意地址不能冲突。

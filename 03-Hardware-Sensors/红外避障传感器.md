# 👀 红外避障传感器 (IR Obstacle Avoidance Sensor)

## 🎯 一句话定义
红外避障传感器就像蚂蚁的触角，用看不见的红外光探测前方是否有障碍物。

## 🧒 给孩子的解释
蚂蚁走路时会用触角探路，碰到东西就知道要转弯。红外避障传感器就是机器人的"触角"！它发出红外光，如果前面有东西，光就会反射回来，机器人就知道"哎呀，前面有障碍，我得绕开！"。比超声波传感器更灵敏，但探测距离更近一些。

## 🔬 工作原理

### 红外反射检测
```
发射管 ──发出红外光──> 障碍物
接收管 <──接收反射光── 障碍物

有障碍物：反射光强 → 输出低电平
无障碍物：反射光弱 → 输出高电平
```

### 与超声波传感器的区别

| 特性 | 红外避障 | 超声波 |
|------|----------|--------|
| 检测方式 | 红外光反射 | 声波反射 |
| 检测距离 | 2-30cm | 2-400cm |
| 检测角度 | 约30° | 约15° |
| 受阳光影响 | 大 | 小 |
| 检测透明物体 | 差 | 好 |
| 响应速度 | 快 | 中等 |
| 成本 | 低 | 中 |

### 核心组件
1. **红外发射管**：发出红外光
2. **红外接收管**：接收反射光
3. **比较器**：判断是否有障碍物
4. **电位器**：调节灵敏度

## 📊 技术参数

| 参数 | 数值 | 说明 |
|------|------|------|
| 工作电压 | 3.3V - 5V | 兼容多种开发板 |
| 工作电流 | 约20mA | 省电 |
| 检测距离 | 2 - 30cm | 可调节 |
| 输出方式 | 数字输出 | 高/低电平 |
| 检测角度 | 约30° | 比超声波宽 |
| 响应时间 | <1ms | 反应快 |
| 指示灯 | LED | 检测到障碍物亮灯 |
| 尺寸 | 约30×15mm | 小巧 |

### 引脚说明
```
VCC  ──── 3.3V或5V电源
GND  ──── 电源负极
OUT  ──── 数字输出（检测到障碍物为LOW）
```

### 灵敏度调节
```
电位器调节方向：
顺时针 → 灵敏度增加，检测距离变远
逆时针 → 灵敏度降低，检测距离变近
```

## 🎮 项目创意

### 入门项目
- **单侧避障**：检测到障碍物就转弯
- **双侧避障**：左右两侧检测，智能避障

### 进阶项目
- **三路避障**：左、中、右三路检测
- **避障+循迹**：在循迹过程中避障
- **迷宫求解**：使用红外避障走迷宫

### 创意项目
- **自动跟随机器人**：保持与障碍物的距离
- **智能垃圾桶**：检测到人手自动开盖
- **防盗报警器**：检测到移动物体报警

### Arduino 接线示例（双路避障）
```
避障模块      Arduino
VCC      ──>  5V
GND      ──>  GND
OUT_LEFT ──>  D2
OUT_RIGHT──>  D3
```

### Micro:bit 接线示例
```
避障模块      Micro:bit
VCC      ──>  3V
GND      ──>  GND
OUT      ──>  P0
```

> 💡 **小贴士**：红外避障传感器在强光下（特别是阳光）会失效，因为阳光中也含有红外光。使用时要注意遮光，或者选择抗干扰能力强的型号。黑色物体的检测效果也较差，因为黑色会吸收红外光。

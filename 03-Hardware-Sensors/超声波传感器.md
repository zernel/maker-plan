# 🦇 超声波传感器 (HC-SR04 Ultrasonic Sensor)

## 🎯 一句话定义
超声波传感器就像蝙蝠的耳朵，发出人耳听不见的声波，通过回声判断物体有多远。

## 🧒 给孩子的解释
想象你在山谷里大喊一声，过一会儿就能听到回声。超声波传感器就是用这个原理工作的！它发出一种我们听不到的声音（就像蝙蝠用的"超声波"），然后等着听回声回来。声音跑得越远，回来的时间就越长。通过计算这个时间，机器人就能知道前面的障碍物有多远了。

## 🔬 工作原理

### 声波发射与接收
```
传感器 ──发出超声波──> 物体
传感器 <──收到回声──── 物体
```

### 核心计算公式
```
距离 = (声速 × 时间) / 2

声速 ≈ 340 m/s（空气中）
时间 = 从发射到接收的时间差
除以2是因为声音来回跑了两趟
```

### 工作流程
1. **发射**：Trig 引脚发送至少 10μs 的高电平触发信号
2. **等待**：传感器自动发送 8 个 40kHz 的超声波脉冲
3. **接收**：Echo 引脚返回高电平，持续时间就是声波往返时间
4. **计算**：用公式得出距离

## 📊 技术参数

| 参数 | 数值 | 说明 |
|------|------|------|
| 工作电压 | 5V DC | 必须5V供电 |
| 静态电流 | <2mA | 很省电 |
| 感应角度 | 15° | 检测范围像一个圆锥 |
| 测量距离 | 2cm - 400cm | 最近2cm，最远4米 |
| 精度 | ±0.3cm | 非常精确 |
| 分辨率 | 0.3cm | 能区分的最小距离 |
| 触发信号 | 10μs TTL | 高电平触发 |
| Echo信号 | TTL电平 | 高电平持续时间=往返时间 |
| 工作频率 | 40kHz | 人耳听不到 |
| 尺寸 | 45×20×15mm | 小巧轻便 |

### 引脚说明
```
VCC  ──── 5V电源正极
Trig ──── 触发引脚（输入）
Echo ──── 回声引脚（输出）
GND  ──── 电源负极
```

## 🎮 项目创意

### 入门项目
- **测距仪**：显示物体距离，学习基础测量
- **倒车雷达**：距离越近，蜂鸣器响得越快

### 进阶项目
- **避障小车**：检测到障碍物自动转弯
- **身高测量器**：挂在墙上测量身高
- **液位检测**：测量水箱水位

### 创意项目
- **手势控制**：通过手的距离控制LED亮度
- **自动喂食器**：宠物靠近时自动投食
- **停车位检测**：检测车位是否有车

## 🔌 兼容性

| 开发板 | 支持 | 说明 |
|--------|------|------|
| Micro:bit | ✅ | 需要扩展板，使用 P0/P1/P2 引脚 |
| Arduino | ✅ | 最常用，直接连接数字引脚 |
| ESP32 | ✅ | 支持，注意3.3V逻辑电平 |
| Raspberry Pi | ✅ | 需要Python库 |

### Arduino 接线示例
```
HC-SR04    Arduino
VCC    ──> 5V
Trig   ──> D9
Echo   ──> D10
GND    ──> GND
```

### Micro:bit 接线示例
```
HC-SR04    Micro:bit (扩展板)
VCC    ──> 3V (实际需要5V，需外接)
Trig   ──> P0
Echo   ──> P1
GND    ──> GND
```

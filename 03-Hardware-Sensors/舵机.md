# 🦾 舵机 (Servo Motor SG90/MG996R)

## 🎯 一句话定义
舵机就像机器人的手臂，能精确指向任何角度，是机器人运动控制的核心。

## 🧒 给孩子的解释
你想让机器人抬起手臂到某个角度吗？舵机就是做这个的！它不像普通电机只会转圈圈，舵机能精确地停在任何角度。就像你的手臂，可以抬到90度、45度，或者任何你想停的位置。机器人跳舞、挥手、抓东西，都离不开舵机！

## 🔬 工作原理

### 内部结构
```
┌─────────────────────────────┐
│  ┌─────┐  ┌─────┐  ┌─────┐ │
│  │ DC  │  │齿轮 │  │电位 │ │
│  │电机 │──│减速 │──│器  │ │
│  └─────┘  └─────┘  └─────┘ │
│         │                   │
│  ┌──────┴──────┐           │
│  │  控制电路    │           │
│  └─────────────┘           │
└─────────────────────────────┘
```

### 工作流程
1. **接收信号**：PWM信号告诉舵机目标角度
2. **比较位置**：控制电路比较目标与当前位置
3. **驱动电机**：根据差异驱动DC电机
4. **减速传动**：齿轮组降低速度，增加扭矩
5. **位置反馈**：电位器检测实际位置
6. **精确停止**：到达目标角度后停止

### PWM控制原理
```
PWM信号周期：20ms（50Hz）
脉冲宽度决定角度：

1ms   ──── 0°（最左）
1.5ms ──── 90°（中间）
2ms   ──── 180°（最右）

     ┌──┐
     │  │
─────┘  └─────────────
 1ms    20ms周期
```

## 📊 技术参数

### SG90 vs MG996R 对比

| 参数 | SG90 | MG996R | 说明 |
|------|------|--------|------|
| 工作电压 | 4.8V-6V | 4.8V-7.2V | 相似 |
| 扭矩 | 1.8kg·cm | 13kg·cm | MG996R更强 |
| 速度 | 0.1s/60° | 0.17s/60° | SG90更快 |
| 重量 | 9g | 55g | SG90更轻 |
| 齿轮材质 | 塑料 | 金属 | MG996R更耐用 |
| 尺寸 | 23×12×29mm | 40×20×43mm | MG996R更大 |
| 角度范围 | 0-180° | 0-180° | 相同 |
| 成本 | ¥3-8 | ¥10-20 | SG90便宜 |
| 适用场景 | 轻载、教学 | 重载、竞赛 | 根据需求选择 |

### 引脚说明
```
棕色线 ──── GND（电源负极）
红色线 ──── VCC（4.8V-6V电源）
橙色线 ──── 信号线（PWM控制）
```

### PWM角度对照表
```
脉冲宽度    角度      占空比(50Hz)
0.5ms      0°        2.5%
1.0ms      45°       5%
1.5ms      90°       7.5%
2.0ms      135°      10%
2.5ms      180°      12.5%
```

### Arduino 接线示例
```
SG90         Arduino
棕线(GND) ──> GND
红线(VCC) ──> 5V
橙线(信号)──> D9（PWM引脚）
```

### Arduino 代码示例
```cpp
#include <Servo.h>

Servo myServo;

void setup() {
  myServo.attach(9);  // 连接到D9引脚
}

void loop() {
  myServo.write(0);    // 转到0度
  delay(1000);
  myServo.write(90);   // 转到90度
  delay(1000);
  myServo.write(180);  // 转到180度
  delay(1000);
}
```

### Micro:bit 接线示例
```
SG90         Micro:bit
棕线(GND) ──> GND
红线(VCC) ──> 3V（或外部5V）
橙线(信号)──> P0
```

### MicroPython 代码示例
```python
from microbit import *
import servo

servo0 = servo.Servo(pin0)

while True:
    servo0.write_angle(0)
    sleep(1000)
    servo0.write_angle(90)
    sleep(1000)
    servo0.write_angle(180)
    sleep(1000)
```

> 💡 **小贴士**：舵机在堵转（被卡住）时会消耗很大电流，可能烧坏舵机或电源。使用时要确保舵机不会被卡住。另外，多个舵机同时运动时电流需求很大，需要外部电源供电，不能只靠开发板供电。SG90适合轻载教学，重载项目要选择MG996R或更强的舵机。

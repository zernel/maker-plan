# 🌡️ 温湿度传感器 (DHT11/DHT22)

## 🎯 一句话定义
温湿度传感器就像一个小小的气象站，能同时测量温度和湿度。

## 🧒 给孩子的解释
你知道天气预报里说的"温度25度，湿度60%"是怎么测出来的吗？就是用温湿度传感器！它就像一个小小的气象站，能同时告诉我们现在有多热、空气有多湿。DHT11适合入门学习，DHT22更精确。用它你可以做一个真正的天气站，或者让风扇自动根据温度调节速度！

## 🔬 工作原理

### DHT11：NTC热敏电阻 + 电阻式湿度传感器
```
温度测量：NTC热敏电阻
        温度↑ → 电阻↓
        
湿度测量：电阻式湿度传感器
        湿度↑ → 电阻↓
```

### DHT22：NTC热敏电阻 + 电容式湿度传感器
```
温度测量：NTC热敏电阻
        温度↑ → 电阻↓
        
湿度测量：电容式湿度传感器
        湿度↑ → 电容↑
```

### 数据传输协议
```
单总线协议（One-Wire）：
主机发送开始信号 → 传感器响应 → 发送40位数据
数据格式：8位湿度整数 + 8位湿度小数 + 8位温度整数 + 8位温度小数 + 8位校验和
```

## 📊 技术参数

### DHT11 vs DHT22 对比

| 参数 | DHT11 | DHT22 | 说明 |
|------|-------|-------|------|
| 工作电压 | 3.3V-5V | 3.3V-5V | 相同 |
| 温度范围 | 0-50°C | -40-80°C | DHT22范围更广 |
| 温度精度 | ±2°C | ±0.5°C | DHT22更精确 |
| 湿度范围 | 20-80%RH | 0-100%RH | DHT22范围更广 |
| 湿度精度 | ±5%RH | ±2%RH | DHT22更精确 |
| 采样间隔 | 1秒 | 2秒 | DHT11更快 |
| 响应时间 | 6-15秒 | 2秒 | DHT22更快 |
| 成本 | ¥3-5 | ¥8-15 | DHT11便宜 |
| 适用场景 | 入门学习 | 精确测量 | 根据需求选择 |

### 引脚说明
```
VCC  ──── 3.3V或5V电源
DATA ──── 数据引脚（需要上拉电阻）
GND  ──── 电源负极
NC   ──── 空脚（部分模块有）
```

### 上拉电阻
```
大多数模块已内置4.7kΩ上拉电阻
如果没有，需要在DATA和VCC之间接一个4.7kΩ电阻
```

## 🎮 项目创意

### 入门项目
- **温度显示器**：在LCD上显示当前温度
- **湿度检测器**：检测空气湿度

### 进阶项目
- **气象站**：温湿度+气压+风速综合监测
- **智能风扇**：根据温度自动调节风速
- **温室监控**：监测植物生长环境

### 创意项目
- **穿衣建议器**：根据温度湿度建议穿什么衣服
- **霉菌预警器**：湿度过高时提醒开窗通风
- **鸡蛋孵化箱**：精确控制孵化温度

### Arduino 接线示例
```
DHT11/DHT22   Arduino
VCC       ──>  5V
DATA      ──>  D2
GND       ──>  GND
```

### Arduino 代码示例
```cpp
#include <DHT.h>

#define DHTPIN 2
#define DHTTYPE DHT11  // 或 DHT22

DHT dht(DHTPIN, DHTTYPE);

void setup() {
  Serial.begin(9600);
  dht.begin();
}

void loop() {
  float humidity = dht.readHumidity();
  float temperature = dht.readTemperature();
  
  Serial.print("温度: ");
  Serial.print(temperature);
  Serial.print("°C  湿度: ");
  Serial.print(humidity);
  Serial.println("%");
  
  delay(2000);
}
```

> 💡 **小贴士**：DHT11和DHT22的引脚顺序可能不同，使用前一定要看清楚模块上的标注。另外，传感器需要远离发热元件（如电机、LED灯带），否则测量的温度会偏高。采样间隔不要太短，否则传感器会发热影响精度。

# ☀️ 光敏电阻 (LDR/Photoresistor)

## 🎯 一句话定义
光敏电阻就像向日葵，能感知光线的强弱，光线越强，电阻越小。

## 🧒 给孩子的解释
你知道向日葵会跟着太阳转吗？因为它能感受到阳光的方向。光敏电阻就像一个小小的"光线感知器"，它能感觉到周围有多亮。光线强的时候，它里面的电阻就变小，电流就容易通过；光线弱的时候，电阻就变大，电流就难通过。这样机器人就能知道现在是白天还是晚上啦！

## 🔬 工作原理

### 光电导效应
```
光照强度 ↑ → 电阻 ↓ → 电流 ↑
光照强度 ↓ → 电阻 ↑ → 电流 ↓
```

### 内部结构
```
┌─────────────────┐
│  ┌───────────┐  │
│  │  光敏材料  │  │  ← 硫化镉(CdS)涂层
│  │  (CdS)    │  │
│  └───────────┘  │
│   ↑         ↑   │
│  电极      电极  │
└─────────────────┘
```

### 工作原理详解
1. **无光照**：电子被束缚，电阻很大（约1MΩ）
2. **有光照**：光子激发电子，电阻变小（约100Ω-1kΩ）
3. **光线越强**：被激发的电子越多，电阻越小

### 电压分压电路
```
VCC ──┬── R1 ──┬── 光敏电阻 ──┬── GND
      │        │              │
      │        └── A0 ────────┘
      │        (模拟输入)
      │
   固定电阻
```

## 📊 技术参数

| 参数 | 数值 | 说明 |
|------|------|------|
| 工作电压 | 3.3V - 5V | 兼容多种开发板 |
| 暗电阻 | 约1MΩ | 完全黑暗时 |
| 亮电阻 | 100Ω - 1kΩ | 强光照射时 |
| 响应时间 | 约20-30ms | 中等速度 |
| 光谱范围 | 400-700nm | 可见光范围 |
| 峰值波长 | 540nm | 绿光附近最灵敏 |
| 温度系数 | 约-0.5%/°C | 温度影响小 |
| 尺寸 | 直径约5mm | 小巧 |

## 🎮 项目创意

### 入门项目
- **光线检测器**：显示当前光照强度
- **自动夜灯**：天黑自动亮灯，天亮自动关灯

### 进阶项目
- **光跟随机器人**：朝着光源方向移动
- **光线报警器**：光线突然变化时报警
- **窗帘控制器**：根据光线自动开关窗帘

### 创意项目
- **植物生长灯**：根据光照自动补光
- **太阳能追踪器**：追踪太阳方向
- **光通信实验**：用光线传输信息

## 🔌 兼容性

| 开发板 | 支持 | 说明 |
|--------|------|------|
| Micro:bit | ✅ | 使用P0-P2读取模拟值 |
| Arduino | ✅ | 使用A0-A5读取模拟值 |
| ESP32 | ✅ | 支持ADC读取 |
| Raspberry Pi | ✅ | 需要ADC模块 |

### Arduino 接线示例
```
光敏电阻      Arduino
一端      ──>  5V
另一端    ──>  A0（模拟输入）
            └── GND（通过10kΩ电阻）
```

### Micro:bit 接线示例
```
光敏电阻      Micro:bit
一端      ──>  3V
另一端    ──>  P0（模拟输入）
            └── GND（通过10kΩ电阻）
```

### Python 代码示例
```python
from microbit import *

while True:
    light = pin0.read_analog()  # 读取模拟值 0-1023
    print("光照强度:", light)
    sleep(100)
```

---

> 💡 **小贴士**：光敏电阻的响应速度较慢（约20-30ms），不适合检测快速变化的光线。如果需要检测快速闪烁的光，应该使用光电二极管或光电三极管。另外，光敏电阻对不同颜色的光灵敏度不同，对绿光最灵敏。

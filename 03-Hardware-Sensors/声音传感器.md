# 🎤 声音传感器 (Sound Sensor)

## 🎯 一句话定义
声音传感器就像机器人的耳朵，能听到声音的大小，让机器人对声音做出反应。

## 🧒 给孩子的解释
你拍手的时候，机器人能听到吗？声音传感器就是机器人的"耳朵"！它能听到周围的声音有多响。声音大的时候，它就给机器人发信号："嘿，有人拍手了！"声音小的时候，它就告诉机器人："周围很安静。"这样你就可以用拍手或者喊声来控制机器人啦！

## 🔬 工作原理

### 驻极体电容麦克风
```
声音波 → 震动膜片 → 电容变化 → 电信号
```

### 工作流程
1. **声波进入**：声音引起空气振动
2. **膜片震动**：麦克风内的膜片跟着震动
3. **电容变化**：膜片与背极之间距离变化，电容跟着变
4. **信号转换**：电容变化转换为电信号
5. **信号放大**：运算放大器放大微弱信号
6. **输出**：模拟输出（声音大小）+ 数字输出（是否超过阈值）

### 双输出说明
```
模拟输出(AO)：0-1023，表示声音大小
数字输出(DO)：HIGH/LOW，表示是否超过设定阈值
```

## 📊 技术参数

| 参数 | 数值 | 说明 |
|------|------|------|
| 工作电压 | 3.3V - 5V | 兼容多种开发板 |
| 工作电流 | 约5mA | 很省电 |
| 频率范围 | 50Hz - 20kHz | 人耳可听范围 |
| 灵敏度 | 可调节 | 电位器调节 |
| 输出方式 | 模拟+数字 | 双输出 |
| 检测距离 | 约1米 | 近距离检测 |
| 指示灯 | LED | 检测到声音亮灯 |
| 尺寸 | 约35×15mm | 小巧 |

### 引脚说明
```
VCC  ──── 3.3V或5V电源
GND  ──── 电源负极
AO   ──── 模拟输出（声音大小）
DO   ──── 数字输出（是否超过阈值）
```

### 灵敏度调节
```
电位器调节：
顺时针 → 灵敏度增加，轻微声音就触发
逆时针 → 灵敏度降低，需要较大声音才触发
```

## 🎮 项目创意

### 入门项目
- **声音检测器**：显示当前声音大小
- **拍手开关**：拍手开灯，再拍手关灯

### 进阶项目
- **噪音监测器**：监测环境噪音水平
- **声音报警器**：检测到异常声音报警
- **声控机器人**：用声音控制机器人行动

### 创意项目
- **音乐节拍器**：跟着音乐节奏闪烁LED
- **语音识别基础**：简单的声音模式识别
- **宠物叫声监测**：监测宠物是否在叫

## 🔌 兼容性

| 开发板 | 支持 | 说明 |
|--------|------|------|
| Micro:bit | ✅ | 使用P0读取模拟值 |
| Arduino | ✅ | 使用A0读取模拟值 |
| ESP32 | ✅ | 支持ADC读取 |
| Raspberry Pi | ✅ | 需要ADC模块 |

### Arduino 接线示例
```
声音传感器    Arduino
VCC      ──>  5V
GND      ──>  GND
AO       ──>  A0（模拟输出）
DO       ──>  D2（数字输出，可选）
```

### Micro:bit 接线示例
```
声音传感器    Micro:bit
VCC      ──>  3V
GND      ──>  GND
AO       ──>  P0（模拟输出）
```

### Python 代码示例
```python
from microbit import *

while True:
    sound = pin0.read_analog()  # 读取模拟值 0-1023
    if sound > 500:  # 声音较大
        display.show(Image.HAPPY)
    else:
        display.show(Image.SAD)
    sleep(100)
```

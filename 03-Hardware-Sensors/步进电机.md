# 🎯 步进电机 (Stepper Motor NEMA 17)

## 🎯 一句话定义
步进电机就像精确的舞者，一步一步移动，能精确控制位置和速度。

## 🧒 给孩子的解释
你见过钟表的秒针吗？它不是一直转，而是"滴答、滴答"一步一步走的。步进电机就像秒针一样，它不是一直转，而是按照固定的步数转动。你可以让它走1步、10步、100步，每一步都精确到位。3D打印机、机器人手臂都用它，因为需要精确定位！

## 🔬 工作原理

### 内部结构
```
┌─────────────────────────────┐
│        转子（永磁体）         │
│        ┌─────┐              │
│        │ N S │              │
│        └─────┘              │
│   ┌──┐    ┌──┐    ┌──┐    │
│   │A │    │B │    │A'│    │  ← 定子线圈
│   │+ │    │+ │    │- │    │
│   └──┘    └──┘    └──┘    │
│        ┌─────┐              │
│        │ S N │              │
│        └─────┘              │
└─────────────────────────────┘
```

### 工作原理
```
两相步进电机：

Step 1: A相+，B相+
        转子对齐A-B方向

Step 2: A相-，B相+
        转子旋转90°

Step 3: A相-，B相-
        转子再旋转90°

Step 4: A相+，B相-
        转子再旋转90°

循环以上步骤，电机连续转动
```

### 步距角
```
NEMA 17常见步距角：1.8°（200步/圈）
整步：1.8°/步
半步：0.9°/步
微步：更小角度（如1/16步=0.1125°）
```

### 驱动方式
```
整步驱动：
  每次给一个线圈通电
  步距角大，扭矩大

半步驱动：
  交替给单线圈和双线圈通电
  步距角减半，运行更平滑

微步驱动：
  线圈电流按正弦波变化
  步距角很小，非常平滑
```

## 📊 技术参数

### NEMA 17 参数

| 参数 | 数值 | 说明 |
|------|------|------|
| 工作电压 | 12V-24V | 常见12V |
| 相电流 | 1.2-1.7A | 根据型号 |
| 步距角 | 1.8° | 200步/圈 |
| 保持扭矩 | 0.2-0.5 N·m | 根据型号 |
| 引线数 | 4线/6线 | 4线最常用 |
| 轴径 | 5mm | 标准尺寸 |
| 安装尺寸 | 42×42mm | NEMA标准 |
| 重量 | 约200-300g | 中等重量 |

### 常见型号对比
```
型号        电流    扭矩      应用
NEMA17-04   1.2A   0.22N·m   轻载
NEMA17-17   1.7A   0.44N·m   常用
NEMA17-23   2.3A   0.55N·m   重载
```

### 引脚说明（4线）
```
A+ ──── 相A正极
A- ──── 相A负极
B+ ──── 相B正极
B- ──── 相B负极
```

### 驱动模块
```
A4988：最常用，最大2A，1/16微步
DRV8825：最大2.5A，1/32微步
TMC2209：静音驱动，最大2A，1/256微步
```

```
A4988          Arduino        步进电机

VMOT     ────  12V电源+

2A       ───────────────────── B+
2B       ───────────────────── B-
```

### Arduino 代码示例
```cpp
#define STEP_PIN 3
#define DIR_PIN 2

void setup() {
  pinMode(STEP_PIN, OUTPUT);
  pinMode(DIR_PIN, OUTPUT);
}

void stepMotor(int steps, bool direction) {
  digitalWrite(DIR_PIN, direction);
  for (int i = 0; i < steps; i++) {
    digitalWrite(STEP_PIN, HIGH);
    delayMicroseconds(1000);
    digitalWrite(STEP_PIN, LOW);
    delayMicroseconds(1000);
  }
}

void loop() {
  stepMotor(200, HIGH);   // 正转一圈（200步）
  delay(1000);
  stepMotor(200, LOW);    // 反转一圈
  delay(1000);
}
```

> 💡 **小贴士**：步进电机在高速时扭矩会下降，选择电机时要留有余量。另外，步进电机长时间通电会发热，不使用时可以通过ENABLE引脚断电。A4988驱动模块的电流限制要根据电机额定电流调节，否则可能烧坏电机或驱动。

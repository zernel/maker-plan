# 🚗 直流减速电机 (DC Gear Motor)

## 🎯 一句话定义
直流减速电机就像玩具车的轮子，转啊转，是机器人移动的动力来源。

## 🧒 给孩子的解释
你玩过遥控车吗？按下前进键，车就往前跑。这里面就是直流减速电机在工作！它里面有小马达转得很快，但是通过齿轮减速后，轮子转得慢了，但是力气变大了。就像你骑自行车，用小齿轮带动大齿轮，上坡就轻松多了！

## 🔬 工作原理

### 电机结构
```
┌─────────────────────────────┐
│  ┌─────┐  ┌─────────────┐  │
│  │ DC  │  │   齿轮组    │  │
│  │电机 │──│（减速增扭） │──│→ 输出轴
│  └─────┘  └─────────────┘  │
└─────────────────────────────┘

DC电机：高速低扭矩
齿轮组：降低速度，增加扭矩
输出轴：低速高扭矩，驱动轮子
```

### 减速原理
```
电机转速：10000 RPM（转/分钟）
减速比：1:100
输出转速：100 RPM = 10000/100

扭矩变化：
输出扭矩 = 电机扭矩 × 减速比 × 效率
```

### H桥控制
```
正转：
  ┌───────────────┐
  │   M           │
  │  + ─ ─ ─ →   │  电流方向：左→右
  │               │
  └───────────────┘

反转：
  ┌───────────────┐
  │   M           │
  │  ← ─ ─ ─ +   │  电流方向：右→左
  │               │
  └───────────────┘

H桥可以控制电流方向，从而控制电机正反转
```

## 📊 技术参数

| 参数 | 数值 | 说明 |
|------|------|------|
| 工作电压 | 3V-12V | 常见6V或12V |
| 空载转速 | 100-300 RPM | 根据减速比 |
| 减速比 | 1:34到1:300+ | 可选 |
| 扭矩 | 0.5-10 kg·cm | 根据型号 |
| 空载电流 | 100-200mA | 中等 |
| 堵转电流 | 1-3A | 很大，要注意 |
| 轴径 | 3mm/6mm | 常见尺寸 |
| 尺寸 | 约30×20mm | 小巧 |

### 常见减速比与转速
```
减速比    空载转速(6V)    特点
1:34     300 RPM        速度快，扭矩小
1:48     200 RPM        平衡选择
1:100    100 RPM        常用
1:200    50 RPM         速度慢，扭矩大
1:300+   30 RPM         极慢，扭矩很大
```

### 引脚说明
```
两根线：
  红线 ──── 正极（接电源正极或H桥输出）
  黑线 ──── 负极（接电源负极或H桥输出）

注意：交换两根线可以改变转向
```

### Arduino + L298N 接线示例
```
L298N          Arduino        电机
ENA  ────────  D5(PWM)       
IN1  ────────  D2            
IN2  ────────  D3            
OUT1 ──────────────────────── 电机红线
OUT2 ──────────────────────── 电机黑线
+12V ────────  电池正极
GND  ────────  电池负极 ──── Arduino GND
```

### Arduino 代码示例
```cpp
#define ENA 5
#define IN1 2
#define IN2 3

void setup() {
  pinMode(ENA, OUTPUT);
  pinMode(IN1, OUTPUT);
  pinMode(IN2, OUTPUT);
}

void loop() {
  // 正转
  digitalWrite(IN1, HIGH);
  digitalWrite(IN2, LOW);
  analogWrite(ENA, 200);  // 速度0-255
  delay(2000);
  
  // 停止
  digitalWrite(IN1, LOW);
  digitalWrite(IN2, LOW);
  delay(1000);
  
  // 反转
  digitalWrite(IN1, LOW);
  digitalWrite(IN2, HIGH);
  analogWrite(ENA, 150);
  delay(2000);
}
```

### Micro:bit + 电机驱动板接线示例
```
电机驱动板     Micro:bit
ENA      ──>  P0（PWM）
IN1      ──>  P1
IN2      ──>  P2
GND      ──>  GND
```

> 💡 **小贴士**：直流电机启动时电流很大（堵转电流），不能直接用开发板的引脚驱动，必须使用电机驱动板（如L298N、TB6612）。另外，电机是感性负载，断电时会产生反向电动势，驱动板上的续流二极管可以保护电路。两个电机的转速可能有微小差异，小车跑偏时需要软件补偿。

# 🎵 蜂鸣器 (Buzzer/Passive Buzzer)

## 🎯 一句话定义
蜂鸣器就像小小的乐器，能演奏任何音符，是机器人发出声音的最简单方式。

## 🧒 给孩子的解释
你会吹口哨吗？蜂鸣器就像机器人的"口哨"！它能发出各种各样的声音。有源蜂鸣器像只会叫的小鸟，一通电就响；无源蜂鸣器像把小提琴，需要你给它节奏才能演奏。用它你可以做一个报警器，或者让机器人演奏一首生日歌！

## 🔬 工作原理

### 有源蜂鸣器 vs 无源蜂鸣器

| 特性 | 有源蜂鸣器 | 无源蜂鸣器 |
|------|------------|------------|
| 内部电路 | 有振荡电路 | 无振荡电路 |
| 控制方式 | 通电就响 | 需要PWM信号 |
| 声音频率 | 固定（约2kHz） | 可调（20Hz-20kHz） |
| 能否演奏音乐 | 不能 | 能 |
| 接线 | 简单 | 需要PWM |
| 成本 | 略贵 | 便宜 |
| 适用场景 | 报警、提示音 | 音乐、音效 |

### 有源蜂鸣器工作原理
```
内部结构：
┌─────────────────┐
│   振荡电路       │  ← 产生固定频率
│   ┌─────┐      │
│   │电磁 │      │  ← 驱动膜片震动
│   │线圈 │      │
│   └─────┘      │
│   ┌─────┐      │
│   │膜片 │      │  ← 发出声音
│   └─────┘      │
└─────────────────┘

通电 → 振荡电路工作 → 电磁线圈驱动膜片 → 发出声音
```

### 无源蜂鸣器工作原理
```
外部PWM信号：
┌─────────────────┐
│   ┌─────┐      │
│   │电磁 │      │  ← PWM信号驱动
│   │线圈 │      │
│   └─────┘      │
│   ┌─────┐      │
│   │膜片 │      │  ← 频率决定音调
│   └─────┘      │
└─────────────────┘

PWM频率改变 → 膜片震动频率改变 → 音调改变
```

## 📊 技术参数

| 参数 | 有源蜂鸣器 | 无源蜂鸣器 | 说明 |
|------|------------|------------|------|
| 工作电压 | 3V-5V | 3V-5V | 相同 |
| 工作电流 | <25mA | <25mA | 相同 |
| 频率 | 固定~2kHz | 20Hz-20kHz | 可调范围 |
| 声压级 | >85dB | >85dB | 相同 |
| 尺寸 | 直径12mm | 直径12mm | 相同 |
| 引脚 | 2个 | 2个 | 相同 |

### 音符频率对照表
```
音符    频率(Hz)    MIDI编号
C4      262         60
D4      294         62
E4      330         64
F4      349         65
G4      392         67
A4      440         69
B4      494         71
C5      523         72
```

### 引脚说明
```
有源蜂鸣器：
  长脚 ──── 正极（VCC）
  短脚 ──── 负极（GND）

无源蜂鸣器：
  长脚 ──── 信号（PWM）
  短脚 ──── 负极（GND）

带+号的是正极
```

## 🎮 项目创意

### 入门项目
- **简单报警器**：通电就响
- **倒计时器**：倒计时结束响铃

### 进阶项目
- **音乐播放器**：演奏简单旋律
- **电子琴**：多个按键对应不同音符
- **报警系统**：检测到异常声音报警

### 创意项目
- **生日歌播放器**：自动演奏生日歌
- **摩尔斯电码**：用声音发送电码
- **声音互动游戏**：根据声音做出反应

### Arduino 接线示例
```
有源蜂鸣器    Arduino
正极      ──>  D8
负极      ──>  GND

无源蜂鸣器    Arduino
信号      ──>  D8（PWM引脚）
负极      ──>  GND
```

### Arduino 代码示例
```cpp
#define BUZZER_PIN 8

// 音符频率
#define NOTE_C4  262
#define NOTE_D4  294
#define NOTE_E4  330
#define NOTE_F4  349
#define NOTE_G4  392
#define NOTE_A4  440
#define NOTE_B4  494
#define NOTE_C5  523

void playNote(int note, int duration) {
  tone(BUZZER_PIN, note, duration);
  delay(duration * 1.3);
}

void setup() {
  // 演奏《小星星》
  playNote(NOTE_C4, 500);
  playNote(NOTE_C4, 500);
  playNote(NOTE_G4, 500);
  playNote(NOTE_G4, 500);
  playNote(NOTE_A4, 500);
  playNote(NOTE_A4, 500);
  playNote(NOTE_G4, 1000);
}

void loop() {
  // 循环播放
}
```

### Micro:bit 接线示例
```
无源蜂鸣器    Micro:bit
信号      ──>  P0
负极      ──>  GND
```

### MicroPython 代码示例
```python
from microbit import *
import music

# 演奏《小星星》
melody = [
    "C4:4", "C4:4", "G4:4", "G4:4",
    "A4:4", "A4:4", "G4:8",
    "F4:4", "F4:4", "E4:4", "E4:4",
    "D4:4", "D4:4", "C4:8"
]

music.play(melody, pin=pin0, wait=True)
```

---

> 💡 **小贴士**：蜂鸣器是感性负载，断电时会产生反向电动势。驱动时最好加一个续流二极管（1N4148）保护电路。另外，蜂鸣器的声音大小与电压有关，5V时最响。多个蜂鸣器同时使用时要注意电流限制。

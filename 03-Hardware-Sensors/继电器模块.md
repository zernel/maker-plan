# ⚡ 继电器模块 (Relay Module)

## 🎯 一句话定义
继电器就像机器人的手，能开关真正的电灯，用小电流控制大电流。

## 🧒 给孩子的解释
你想让机器人帮你开关电灯吗？继电器就是机器人的"手"！它就像一个遥控开关，用小小的电流（比如Arduino的5V）就能控制大大的电流（比如家里的220V电灯）。就像你用遥控器开关电视一样，机器人用继电器开关电器！

## 🔬 工作原理

### 电磁继电器结构
```
┌─────────────────────────────┐
│   ┌───────────────────┐    │
│   │    电磁线圈        │    │  ← 通电产生磁力
│   └───────────────────┘    │
│           │                │
│   ┌───────┴───────┐       │
│   │    衔铁        │       │  ← 被磁力吸引
│   └───────┬───────┘       │
│           │                │
│   ┌───────┴───────┐       │
│   │    触点        │       │  ← 开关触点
│   └───────────────┘       │
└─────────────────────────────┘
```

### 工作流程
```
控制端（低压）：
  Arduino输出HIGH → 光耦导通 → 继电器线圈通电

被控端（高压）：
  线圈通电 → 产生磁力 → 吸合衔铁 → 触点闭合 → 电器通电

断开：
  Arduino输出LOW → 光耦截止 → 继电器线圈断电
  线圈断电 → 磁力消失 → 衔铁弹回 → 触点断开 → 电器断电
```

### 光耦隔离
```
控制端与被控端完全隔离：
  ┌─────────────┐    ┌─────────────┐
  │   控制电路   │    │   被控电路   │
  │   (5V DC)   │    │  (220V AC)  │
  │      │      │    │      │      │
  │   ┌──┴──┐  │    │  ┌──┴──┐   │
  │   │光耦 │  │    │  │触点 │   │
  │   │LED │──│────│──│    │   │
  │   └──┬──┘  │    │  └──┬──┘   │
  │      │      │    │      │      │
  └─────────────┘    └─────────────┘

光耦通过光信号传输，电气完全隔离
保护低压电路不受高压干扰
```

## 📊 技术参数

| 参数 | 数值 | 说明 |
|------|------|------|
| 控制电压 | 3.3V-5V | 兼容多种开发板 |
| 控制电流 | 约5mA | 很省电 |
| 触点容量 | 10A/250VAC | 常见规格 |
| 触点类型 | 常开(NO)/常闭(NC) | 可选 |
| 工作温度 | -40°C到85°C | 宽温度范围 |
| 响应时间 | 约10ms | 中等速度 |
| 寿命 | 10万次以上 | 长寿命 |
| 尺寸 | 约50×25mm | 中等大小 |

### 触点类型
```
常开(NO)：默认断开，通电闭合
  未通电：电路断开
  通电后：电路闭合

常闭(NC)：默认闭合，通电断开
  未通电：电路闭合
  通电后：电路断开

公共端(COM)：公共连接点
```

### 引脚说明
```
控制端：
  VCC  ──── 3.3V或5V电源
  GND  ──── 电源负极
  IN   ──── 控制信号（HIGH触发）

被控端：
  COM  ──── 公共端
  NO   ──── 常开端
  NC   ──── 常闭端
```

### Arduino 接线示例
```
继电器模块      Arduino
VCC        ──>  5V
GND        ──>  GND
IN         ──>  D7

被控电路：
COM        ──>  电源火线
NO         ──>  电器火线
电器零线   ──>  电源零线
```

### Arduino 代码示例
```cpp
#define RELAY_PIN 7

void setup() {
  pinMode(RELAY_PIN, OUTPUT);
}

void loop() {
  // 开灯
  digitalWrite(RELAY_PIN, HIGH);
  delay(2000);
  
  // 关灯
  digitalWrite(RELAY_PIN, LOW);
  delay(2000);
}
```

```
继电器模块      Micro:bit

IN         ──>  P0
```

---

> ⚠️ **安全警告**：继电器模块的被控端涉及高压电路（220V），操作不当可能导致触电或火灾！

### 安全注意事项
1. **必须在断电状态下接线**
2. **被控电路必须有保险丝保护**
3. **高压线路必须绝缘处理**
4. **儿童操作必须有成人陪同**
5. **建议使用12V/24V低压项目练习**

> 💡 **小贴士**：继电器是机械部件，有寿命限制（约10万次）。频繁开关的场景建议使用固态继电器（SSR）。另外，继电器线圈断电时会产生反向电动势，需要续流二极管保护电路。

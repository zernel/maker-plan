# 🌱 土壤湿度传感器 (Soil Moisture Sensor)

## 🎯 一句话定义
土壤湿度传感器就像植物的医生，能检查土壤是否"口渴"，需要浇水。

## 🧒 给孩子的解释
你养过植物吗？植物渴了会蔫掉，但等它蔫了再浇水就太晚了！土壤湿度传感器就是植物的"医生"，它能提前告诉你土壤是不是干了。把它插进土里，它就能感觉到土壤里有多少水分。水分多的时候，它告诉机器人"植物不渴"；水分少的时候，它说"快浇水！"这样你的植物就永远不会渴死啦！

## 🔬 工作原理

### 两种类型对比

#### 电阻式土壤湿度传感器
```
原理：测量土壤导电性
      水分多 → 导电性好 → 电阻低 → 电压高
      水分少 → 导电性差 → 电阻高 → 电压低

优点：便宜，响应快
缺点：容易腐蚀，寿命短（几个月）
```

#### 电容式土壤湿度传感器
```
原理：测量土壤介电常数
      水分多 → 介电常数大 → 电容大
      水分少 → 介电常数小 → 电容小

优点：不易腐蚀，寿命长（几年）
缺点：稍贵，响应稍慢
```

### 传感器结构
```
电阻式：
┌─────────────┐
│    PCB板     │
│   ┌─────┐   │
│   │ 电路 │   │
│   └─────┘   │
└──────┬──────┘
       │
    ┌──┴──┐
    │叉子 │  ← 插入土壤
    │     │
    └─────┘

电容式：
┌─────────────┐
│    PCB板     │
│   ┌─────┐   │
│   │电路 │   │
│   └─────┘   │
└──────┬──────┘
       │
    ┌──┴──┐
    │薄片 │  ← 插入土壤
    │     │
    └─────┘
```

## 📊 技术参数

| 参数 | 电阻式 | 电容式 | 说明 |
|------|--------|--------|------|
| 工作电压 | 3.3V-5V | 3.3V-5V | 相同 |
| 工作电流 | 约20mA | 约5mA | 电容式更省电 |
| 输出方式 | 模拟+数字 | 模拟 | 都可使用 |
| 测量范围 | 0-100% | 0-100% | 相同 |
| 精度 | ±5% | ±3% | 电容式更准 |
| 响应时间 | <1s | <1s | 相同 |
| 寿命 | 几个月 | 几年 | 电容式更长 |
| 成本 | ¥1-3 | ¥3-8 | 电阻式便宜 |
| 尺寸 | 约60×20mm | 约60×20mm | 相似 |

### 引脚说明（电阻式）
```
VCC  ──── 3.3V或5V电源
GND  ──── 电源负极
AO   ──── 模拟输出（湿度值）
DO   ──── 数字输出（是否超过阈值）
```

### 引脚说明（电容式）
```
VCC  ──── 3.3V或5V电源
GND  ──── 电源负极
AO   ──── 模拟输出（湿度值）
```

## 🎮 项目创意

### 入门项目
- **土壤湿度检测**：显示当前土壤湿度
- **浇水提醒器**：湿度低于阈值时提醒

### 进阶项目
- **自动浇水系统**：湿度低时自动浇水
- **智能花园**：多盆植物监测与自动浇水
- **植物生长监测**：记录土壤湿度变化曲线

### 创意项目
- **远程植物监控**：通过WiFi远程查看湿度
- **多肉植物管家**：为不同植物设定不同湿度阈值
- **校园气象站**：结合温湿度传感器综合监测

### Arduino 接线示例
```
土壤湿度传感器  Arduino
VCC        ──>  5V
GND        ──>  GND
AO         ──>  A0（模拟输出）
DO         ──>  D2（数字输出，可选）
```

### Arduino 代码示例
```cpp
#define SENSOR_PIN A0

void setup() {
  Serial.begin(9600);
}

void loop() {
  int moisture = analogRead(SENSOR_PIN);
  int percent = map(moisture, 1023, 0, 0, 100);  // 转换为百分比
  
  Serial.print("土壤湿度: ");
  Serial.print(percent);
  Serial.println("%");
  
  if (percent < 30) {
    Serial.println("土壤太干，需要浇水！");
  }
  
  delay(1000);
}
```

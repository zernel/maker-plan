# 🌈 颜色传感器 (TCS3200 Color Sensor)

## 🎯 一句话定义
颜色传感器就像超级颜色视觉，能精确分辨红、绿、蓝三种颜色的强度。

## 🧒 给孩子的解释
你的眼睛能分辨红苹果和绿苹果，对吧？颜色传感器就是机器人的"颜色眼睛"！但它比人眼更厉害，它能把颜色拆成红、绿、蓝三种光来测量，每种光有多强它都知道。这样机器人就能分清不同颜色的东西，比如把红色积木和蓝色积木分开，或者跟着彩色线走！

## 🔬 工作原理

### 颜色识别原理
```
白光 = 红光 + 绿光 + 蓝光

颜色传感器内部：
┌─────────────────────┐
│  ┌───┬───┬───┬───┐  │
│  │ R │ G │ B │ R │  │  ← 8×8光电二极管阵列
│  ├───┼───┼───┼───┤  │    (16个红色滤光片,
│  │ G │ B │ R │ G │  │     16个绿色滤光片,
│  ├───┼───┼───┼───┤  │     16个蓝色滤光片,
│  │ B │ R │ G │ B │  │     16个无滤光片)
│  ├───┼───┼───┼───┤  │
│  │ R │ G │ B │ R │  │
│  └───┴───┴───┴───┘  │
└─────────────────────┘
```

### 工作流程
1. **选择滤光片**：通过S2/S3引脚选择红、绿、蓝滤光片
2. **光电转换**：光电二极管将光转换为电流
3. **频率转换**：电流转换为方波频率
4. **测量频率**：频率越高，对应颜色越强

### 颜色判断
```
红色物体：红光频率高，绿光和蓝光频率低
绿色物体：绿光频率高，红光和蓝光频率低
蓝色物体：蓝光频率高，红光和绿光频率低
白色物体：三种光频率都高
黑色物体：三种光频率都低
```

## 📊 技术参数

| 参数 | 数值 | 说明 |
|------|------|------|
| 工作电压 | 3.3V - 5V | 兼容多种开发板 |
| 工作电流 | 约2.5mA | 很省电 |
| 输出方式 | 频率输出 | 方波 |
| 频率范围 | 10Hz - 12kHz | 可选分频 |
| 检测距离 | 约1cm | 需要近距离 |
| 响应时间 | 约100ms | 中等速度 |
| 尺寸 | 约33×22mm | 小巧 |

### 引脚说明
```
VCC  ──── 3.3V或5V电源
GND  ──── 电源负极
S0   ──── 频率分频选择
S1   ──── 频率分频选择
S2   ──── 滤光片选择
S3   ──── 滤光片选择
OUT  ──── 频率输出
OE   ──── 输出使能（可选）
```

### 频率分频设置
```
S0   S1    输出频率
LOW  LOW   关闭
LOW  HIGH  2%
HIGH LOW   20%
HIGH HIGH  100%
```

### 滤光片选择
```
S2   S3    滤光片颜色
LOW  LOW   红色
LOW  HIGH  蓝色
HIGH LOW   无滤光片
HIGH HIGH  绿色
```

## 🎮 项目创意

### 入门项目
- **颜色识别器**：识别并显示当前颜色
- **颜色分类器**：将不同颜色的物体分类

### 进阶项目
- **颜色分拣机器人**：自动分拣不同颜色的物品
- **颜色循迹**：沿着彩色线条行驶
- **颜色匹配游戏**：按颜色匹配物体

### 创意项目
- **调色板分析器**：分析画作中的颜色组成
- **肤色检测器**：用于美容或健康检测
- **颜色翻译器**：将颜色转换为声音或振动

### Arduino 接线示例
```
TCS3200      Arduino
VCC      ──>  5V
GND      ──>  GND
S0       ──>  D4
S1       ──>  D5
S2       ──>  D6
S3       ──>  D7
OUT      ──>  D8
```

### Arduino 代码示例
```cpp
#define S0 4
#define S1 5
#define S2 6
#define S3 7
#define OUT 8

void setup() {
  pinMode(S0, OUTPUT);
  pinMode(S1, OUTPUT);
  pinMode(S2, OUTPUT);
  pinMode(S3, OUTPUT);
  pinMode(OUT, INPUT);
  
  // 设置频率分频为100%
  digitalWrite(S0, HIGH);
  digitalWrite(S1, HIGH);
  
  Serial.begin(9600);
}

int readColor(int s2Val, int s3Val) {
  digitalWrite(S2, s2Val);
  digitalWrite(S3, s3Val);
  delay(100);
  return pulseIn(OUT, LOW);
}

void loop() {
  int red = readColor(LOW, LOW);      // 红色
  int green = readColor(HIGH, HIGH);  // 绿色
  int blue = readColor(LOW, HIGH);    // 蓝色
  
  Serial.print("R:"); Serial.print(red);
  Serial.print(" G:"); Serial.print(green);
  Serial.print(" B:"); Serial.println(blue);
  
  delay(500);
}
```

> 💡 **小贴士**：颜色传感器对环境光很敏感，使用时要用遮光罩包围传感器和被测物体，避免环境光干扰。另外，被测物体表面要平整，距离传感器约1cm效果最好。深色物体的测量精度会比浅色物体低。

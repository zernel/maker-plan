### 1. 全国青少年人工智能创新挑战赛

| 项目 | 内容 |
|------|------|
| **白名单序号** | 第1项 |
| **主办单位** | 中国少年儿童发展服务中心 |
| **面向学段** | 小学全学段、初中、高中 |
| **官方网站** | https://www.aicc.org.cn |
| **报名方式** | 官网在线报名 |
| **报名费用** | 免费 |
| **竞赛形式** | 个人赛/团队赛（2-3人） |
| **竞赛内容** | AI创意编程、智能机器人、AI应用开发等 |

#### 时间节点（2027年预计）

```
2027年1月  ──── 赛事规则发布
2027年3月  ──── 赛事说明会
2027年5月  ──── 报名截止
2027年6月  ──── 初赛/省赛
2027年7月  ──── 全国决赛报名
2027年8月  ──── 全国决赛
2027年9月  ──── 成绩公布
```

#### 备赛建议

- **1-2月**：研究赛事规则，确定参赛方向
- **3-4月**：基础技能训练，完成1-2个练手项目
- **5月**：确定参赛作品，开始开发
- **6-7月**：作品打磨，准备初赛
- **8月**：决赛准备，练习答辩

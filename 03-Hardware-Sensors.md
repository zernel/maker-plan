## 📋 使用指南

### 如何选择传感器？

```
孩子年龄 → 项目需求 → 选择传感器

4-6岁：触摸、声音、LED（简单互动）
7-9岁：超声波、光敏、温度（测量类）
10-12岁：IMU、摄像头、通信模块（复杂系统）
```

## 📚 十一、项目创意速查

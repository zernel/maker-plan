# 📖 开源课程资源

> **核心价值**：优质的开源课程资源能大幅降低学习成本，让孩子在家也能系统学习。
> **筛选标准**：免费或低成本、有中文支持、内容质量高、适合中国孩子

## 📋 资源使用建议

### 按学习阶段选择

| 阶段 | 推荐资源 | 重点 |
|------|----------|------|
| **启蒙期（4-6岁）** | Code.org、CS Unplugged | 不插电活动 |
| **入门期（6-8岁）** | Scratch、micro:bit官方 | 图形化编程 |
| **进阶期（8-10岁）** | Arduino、DFRobot | 硬件结合 |
| **提高期（10-12岁）** | Python、Raspberry Pi | 文本编程 |
| **竞赛期（12岁+）** | CMU、Stanford AI4K12 | 系统学习 |

### 按学习方式选择

| 方式 | 推荐资源 |
|------|----------|
| **自学** | Code.org、Scratch官网、Arduino教程 |
| **家长辅导** | CS Unplugged、micro:bit项目 |
| **机构学习** | LEGO Education、Makeblock课程 |
| **竞赛准备** | CMU、竞赛真题 |

### 资源组合建议

**免费组合**：
- Code.org + Scratch + Arduino官方教程

**低成本组合**：
- micro:bit套件 + DFRobot教程 + CS Unplugged

**系统学习组合**：
- LEGO课程 + 专业培训机构 + 竞赛社区

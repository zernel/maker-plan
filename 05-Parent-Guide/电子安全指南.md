# ⚡ 电子安全指南

> **核心原则**：安全是创客教育的第一课。了解风险，才能安全地探索。
> **参考标准**：IEC 62115（电动玩具安全）、GB/T 33265-2025（儿童电子教育产品安全）、NFPA电气安全指南

## 🔋 电压安全：年龄与电压限制对照表

电压是电子安全中最基本的参数。不同年龄段的孩子应接触不同电压等级的设备。

### 电压等级与年龄对照

| 电压等级 | 安全分类 | 适合年龄 | 常见设备 | 注意事项 |
|----------|----------|----------|----------|----------|
| **≤3V** | 🟢 极低风险 | 4岁+ | 纽扣电池、LED小灯 | 仍需防误吞 |
| **3-5V** | 🟢 低风险 | 6岁+ | micro:bit（3V）、Arduino（5V） | USB供电为主 |
| **5-12V** | 🟡 中等风险 | 8岁+ | 电机驱动、传感器模块 | 需要大人监督 |
| **12-24V** | 🟠 较高风险 | 10岁+ | 部分机器人套件 | 同室监督 |
| **>24V** | 🔴 高风险 | 12岁+且有培训 | 专业设备 | 全程监督 |

### 关键安全规则

1. **永远不要接触家用电压（220V）**，这对成人也是危险的
2. **电池供电是最安全的选择**，优先选择USB充电设备
3. **不确定电压时，先问大人**
4. **湿手不要碰任何电子设备**

## 🔋 电池安全矩阵

电池是创客项目中最常用的供电方式，但不同类型的电池有不同的安全要求。

### 电池类型对照

| 电池类型 | 电压 | 适合年龄 | 安全等级 | 使用场景 |
|----------|------|----------|----------|----------|
| **AA/AAA干电池** | 1.5V | 4岁+ | 🟢 安全 | 入门套件、玩具 |
| **9V方块电池** | 9V | 8岁+ | 🟡 注意 | 简单电路实验 |
| **纽扣电池(CR2032)** | 3V | 6岁+（需防误吞） | 🟡 注意 | 小型LED项目 |
| **锂电池(3.7V)** | 3.7V | 10岁+ | 🟠 监督 | 机器人、无人机 |
| **锂电池组(7.4V+)** | 7.4V+ | 12岁+ | 🔴 高风险 | 高级机器人 |

### 锂电池特别警告 ⚠️

锂电池能量密度高，使用不当可能引发火灾或爆炸：

**绝对禁止**：
- ❌ 用针刺穿电池
- ❌ 将电池投入火中
- ❌ 过度充电（使用劣质充电器）
- ❌ 正负极短路
- ❌ 使用鼓包/变形的电池

**正确做法**：
- ✅ 使用原装或品牌充电器
- ✅ 充电时有人看管
- ✅ 存放在阴凉干燥处
- ✅ 不用时断开连接
- ✅ 定期检查电池状态

### 电池存储建议

```
✅ 正确存储：
- 干燥阴凉处（避免高温）
- 远离金属物品（防短路）
- 正负极分开存放
- 使用电池盒

❌ 错误存储：
- 阳光直射处
- 口袋里混放
- 潮湿环境
- 与金属钥匙混放
```

## 🛠️ 工具安全：年龄适宜工具清单

不同年龄段的孩子应使用不同的工具。选择适合的工具，既能完成项目，又能保证安全。

### 工具与年龄对照

| 工具 | 适合年龄 | 安全提示 |
|------|----------|----------|
| **塑料螺丝刀** | 4岁+ | 无尖锐边缘 |
| **儿童安全剪刀** | 5岁+ | 圆头设计 |
| **烙铁（低温）** | 10岁+ | 需要培训和监督 |
| **热风枪** | 12岁+ | 成人操作为主 |
| **万用表** | 8岁+ | 教会正确使用方法 |
| **剥线钳** | 8岁+ | 选择儿童尺寸 |
| **尖嘴钳** | 8岁+ | 注意夹伤风险 |
| **小型电钻** | 12岁+ | 成人监督 |

### 工具使用基本原则

1. **用前检查**：每次使用前检查工具是否完好
2. **正确握持**：教会孩子正确的握持方式
3. **用后归位**：养成工具用完放回原处的习惯
4. **定期维护**：保持工具清洁和锋利（钝的工具更危险）

## 🔥 焊接安全：何时可以开始

焊接是创客项目中的重要技能，但涉及高温和有害气体，需要特别注意。

### 焊接适宜年龄

| 年龄 | 建议 |
|------|------|
| **<10岁** | ❌ 不建议学习焊接 |
| **10-12岁** | ⚠️ 可以开始接触，需全程监督 |
| **12岁+** | ✅ 可以独立操作，定期检查 |

### 焊接必备装备

| 装备 | 用途 | 价格参考 |
|------|------|----------|
| **防静电焊台** | 温度可控，更安全 | ¥100-300 |
| **焊接手套** | 防烫伤 | ¥20-50 |
| **护目镜** | 保护眼睛 | ¥30-80 |
| **排烟风扇** | 吸走有害气体 | ¥50-150 |
| **焊接垫** | 保护桌面 | ¥20-40 |
| **助焊剂** | 提高焊接质量 | ¥10-30 |

### 焊接安全操作流程

```
1. 准备阶段
   ├── 检查焊台温度（通常300-380°C）
   ├── 确保通风良好
   ├── 戴好护目镜和手套
   └── 清理工作区域

2. 焊接阶段
   ├── 焊接时不要触摸烙铁头
   ├── 不要将烙铁放在易燃物上
   ├── 焊接时保持稳定姿势
   └── 避免吸入焊烟

3. 收尾阶段
   ├── 关闭焊台电源
   ├── 等待烙铁冷却
   ├── 清理焊锡渣
   └── 保存好工具
```

### 焊接替代方案

如果孩子还小或家长不放心，可以考虑：
- **面包板**：免焊接，可重复使用
- **导电胶带**：适合简单电路
- **压接连接器**：无需焊接的连接方式
- **焊接练习板**：专门用于练习的板子

## ⚙️ 元件安全

不同电子元件有不同的安全注意事项。

### 电机

| 风险 | 预防措施 |
|------|----------|
| 夹伤手指 | 使用带保护罩的电机 |
| 过热 | 不要长时间堵转 |
| 噪音 | 控制运行时间 |

### 继电器

| 风险 | 预防措施 |
|------|----------|
| 电弧 | 不要切换高电压 |
| 误触发 | 编程时设置安全逻辑 |

### LED

| 风险 | 预防措施 |
|------|----------|
| 亮度刺眼 | 不要直视高亮LED |
| 过热 | 大功率LED需要散热 |
| 电流过大 | 串联限流电阻 |

### 化学品

| 物品 | 风险 | 预防措施 |
|------|------|----------|
| 助焊剂 | 刺激呼吸道 | 通风环境使用 |
| 热熔胶 | 烫伤 | 使用低温胶枪 |
| 清洗剂 | 皮肤刺激 | 戴手套 |

## 🏠 工作空间设置

一个好的工作空间能大大提高安全性。

### 照明

- **主照明**：均匀照亮整个工作区域
- **局部照明**：台灯用于精细操作
- **避免反光**：使用哑光桌面

### 通风

- **自然通风**：开窗是最简单有效的方式
- **排风扇**：焊接时必须使用
- **空气净化器**：辅助去除微粒

### 存储

```
推荐存储方案：
├── 元件收纳盒（分类存放）
├── 工具挂架（方便取用）
├── 电池专用盒（防短路）
└── 项目展示架（展示成果）
```

### 桌面布局

## 👨‍👩‍👧‍👦 监督指南

不同年龄段需要不同程度的监督。

### 监督级别对照

| 年龄 | 监督级别 | 具体要求 |
|------|----------|----------|
| **4-6岁** | 🟢 全程陪同 | 大人始终在旁，手把手指导 |
| **7-9岁** | 🟡 同室监督 | 大人在同一房间，定期查看 |
| **10-12岁** | 🟠 定期检查 | 大人每15-20分钟查看一次 |
| **12岁+** | 🔵 自主操作 | 独立完成，有问题找大人 |

### 监督要点

**全程陪同阶段（4-6岁）**：
- 所有工具由大人保管和递送
- 电池由大人安装和取出
- 每一步都由大人示范后孩子模仿

**同室监督阶段（7-9岁）**：
- 孩子可以自己取用工具
- 大人需要确认操作是否正确
- 定期询问"需要帮忙吗？"

**定期检查阶段（10-12岁）**：
- 孩子可以独立完成大部分操作
- 大人检查关键步骤（如接线、通电）
- 项目完成后一起复盘

## 🚑 常见伤害预防

### 烧伤

**常见原因**：烙铁、热熔胶枪、发热元件

**预防措施**：
- 使用低温烙铁（可调温）
- 热熔胶枪选择低温款
- 发热元件加散热片
- 准备急救烫伤药膏

**应急处理**：
1. 立即用冷水冲洗10-15分钟
2. 不要涂抹牙膏等偏方
3. 严重时及时就医

### 割伤

**常见原因**：剪刀、刀片、金属边缘

**预防措施**：
- 使用圆头剪刀
- 金属板材边缘打磨
- 操作时注意力集中

**应急处理**：
1. 用清水冲洗伤口
2. 用创可贴或纱布包扎
3. 伤口较深时就医

### 眼疲劳

**常见原因**：长时间看屏幕、焊接弧光

**预防措施**：
- 遵守20-20-20法则
- 焊接时戴护目镜
- 保持适当光线

**应急处理**：
1. 闭眼休息
2. 远眺放松
3. 持续不适时就医

### 触电

**常见原因**：接触裸露导线、短路

**预防措施**：
- 使用绝缘工具
- 通电前检查接线
- 湿手不碰电路

**应急处理**：
1. 立即切断电源
2. 不要直接接触触电者
3. 严重时立即拨打120

## 📋 安全检查清单

每次项目开始前，对照检查：

- [ ] 工作区域整洁，无杂物
- [ ] 工具完好，无损坏
- [ ] 电池电量充足，无鼓包
- [ ] 通风良好
- [ ] 照明充足
- [ ] 急救包在附近
- [ ] 了解紧急联系方式

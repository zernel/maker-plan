# 🧒 Track A: 4岁幼儿园启蒙路径

> **适合年龄**: 4-5岁幼儿园阶段
> **学习周期**: 6-12个月（每周1-2次，每次20-30分钟）
> **核心理念**: 在玩耍中建立编程思维基础

---

## 🎯 核心目标

| 维度 | 目标 |
|------|------|
| 🧠 **逻辑思维** | 理解顺序、循环、条件判断的基本概念 |
| 🖐️ **动手能力** | 大颗粒积木搭建、简单机械结构 |
| 🗣️ **表达能力** | 能用语言描述"先...然后...最后..." |
| 👀 **空间感知** | 理解前后左右上下方向 |

---

## 🛠️ 推荐器材

### 主力器材
- **乐高 Coding Express** - 火车编程启蒙，通过放置动作块控制火车行为
- **玛塔创客 (Matalab)** - 实体编程机器人，无需屏幕即可编程

### 辅助器材
- 大颗粒积木套装（齿轮、滑轮、杠杆组件）
- 方向骰子、排序卡片
- 不插电编程活动材料包

## 🎮 教学方法

### 不插电活动优先
这个阶段的孩子不需要屏幕。通过身体游戏、积木搭建、故事讲述来建立概念。

参考: [[02-Concepts/不插电编程]]

### 做中学
每个概念都通过动手实验来理解，而不是讲解理论。

参考: [[02-Concepts/做中学]]

### 正向鼓励
关注过程而非结果，鼓励孩子用自己的方式解决问题。

## 📝 家长提示

1. **不要急**: 4岁孩子的注意力有限，20分钟就够了
2. **多重复**: 同一个实验可以玩很多次，每次都有新发现
3. **少纠正**: 让孩子自己发现错误比直接告诉更好
4. **多提问**: "你觉得接下来会发生什么？"比"这样做"更好

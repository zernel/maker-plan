## 🧠 核心原理简述（给家长）

**状态机 (Finite State Machine, FSM)** 是一种描述系统行为的数学模型。用大白话说，就是让程序在不同"状态"之间切换。

生活中的状态机例子：
- **红绿灯**：红灯 → 绿灯 → 黄灯 → 红灯（循环）
- **自动售货机**：等待投币 → 选择商品 → 出货 → 找零
- **洗衣机**：进水 → 洗涤 → 排水 → 脱水 → 完成

为什么学状态机：
- **管理复杂行为**：机器人同时做多件事时，状态机让逻辑清晰
- **竞赛必备**：WRC、MakeX等赛事的机器人需要状态机来切换任务
- **调试友好**：出问题时，能快速定位是哪个状态出了错
- **代码整洁**：比一堆 if-else 嵌套好维护得多

状态机的三个核心要素：
1. **状态 (State)**：系统当前处于什么模式（如"巡逻中"、"避障中"）
2. **事件 (Event)**：触发状态切换的条件（如"检测到障碍物"）
3. **转移 (Transition)**：从一个状态切换到另一个状态

- 220Ω电阻 × 3

**玩法描述**:
1. 用状态机实现红绿灯：
```python
from machine import Pin
import time

# 定义LED引脚
red_led = Pin(2, Pin.OUT)
yellow_led = Pin(4, Pin.OUT)
green_led = Pin(5, Pin.OUT)

# 定义状态
RED = 0
GREEN = 1
YELLOW = 2

# 初始状态
current_state = RED

def set_lights(r, y, g):
    red_led.value(r)
    yellow_led.value(y)
    green_led.value(g)

while True:
    if current_state == RED:
        set_lights(1, 0, 0)  # 红灯亮
        print("状态：红灯 - 停！")
        time.sleep(3)
        current_state = GREEN  # 转移到绿灯状态
        
    elif current_state == GREEN:
        set_lights(0, 0, 1)  # 绿灯亮
        print("状态：绿灯 - 行！")
        time.sleep(3)
        current_state = YELLOW  # 转移到黄灯状态
        
    elif current_state == YELLOW:
        set_lights(0, 1, 0)  # 黄灯亮
        print("状态：黄灯 - 注意！")
        time.sleep(1)
        current_state = RED  # 转移到红灯状态
```

### 实验2: 🤖 机器人巡逻状态机

**零件清单**:
- ESP32 或 Arduino Nano
- 超声波传感器 HC-SR04 × 1
- 舵机 SG90 × 1（用于转向）
- 红外循迹传感器 × 2
- LED × 2（状态指示）
- 面包板 × 1
- 杜邦线若干
- 小车底盘（可选）

**玩法描述**:
1. 创建一个巡逻机器人，有三种状态：
```python
from machine import Pin, PWM
import time

# 硬件初始化
trig = Pin(12, Pin.OUT)
echo = Pin(13, Pin.IN)
servo = PWM(Pin(15), freq=50)
left_ir = Pin(16, Pin.IN)
right_ir = Pin(17, Pin.IN)
status_led = Pin(2, Pin.OUT)

# 状态定义
PATROL = 0      # 巡逻状态
OBSTACLE = 1    # 避障状态
LINE_FOLLOW = 2 # 循迹状态

current_state = PATROL
obstacle_count = 0

def get_distance():
    trig.value(0)
    time.sleep_us(2)
    trig.value(1)
    time.sleep_us(10)
    trig.value(0)
    while echo.value() == 0:
        start = time.ticks_us()
    while echo.value() == 1:
        end = time.ticks_us()
    return (end - start) / 58

def patrol():
    print("巡逻中... 搜索目标")
    servo.duty(77)  # 直行
    status_led.value(1)
    time.sleep(0.5)

def avoid_obstacle():
    global obstacle_count
    print("检测到障碍物！开始避障")
    servo.duty(50)  # 转向
    time.sleep(1)
    obstacle_count += 1
    print(f"已避开 {obstacle_count} 个障碍物")

def follow_line():
    left = left_ir.value()
    right = right_ir.value()
    if left == 0 and right == 0:
        print("循迹：直行")
        servo.duty(77)
    elif left == 1 and right == 0:
        print("循迹：左转")
        servo.duty(60)
    elif left == 0 and right == 1:
        print("循迹：右转")
        servo.duty(90)

# 主循环
while True:
    distance = get_distance()
    
    if current_state == PATROL:
        patrol()
        # 检测是否需要切换状态
        if distance < 20:
            current_state = OBSTACLE
        elif left_ir.value() == 0 or right_ir.value() == 0:
            current_state = LINE_FOLLOW
            
    elif current_state == OBSTACLE:
        avoid_obstacle()
        # 避障完成后回到巡逻
        if distance > 30:
            current_state = PATROL
            
    elif current_state == LINE_FOLLOW:
        follow_line()
        # 循迹中断，回到巡逻
        if distance < 20:
            current_state = OBSTACLE
        elif left_ir.value() == 1 and right_ir.value() == 1:
            current_state = PATROL
```

### 实验3: 🎮 用类实现状态机

**零件清单**:
- 电脑（安装 Python 3.8+）
- ESP32（可选，用于硬件测试）

**玩法描述**:
1. 用面向对象的方式重构状态机：
```python
class StateMachine:
    def __init__(self, name):
        self.name = name
        self.current_state = None
        self.states = {}
        self.transitions = {}
    
    def add_state(self, state_name, on_enter=None, on_exit=None):
        """添加状态"""
        self.states[state_name] = {
            'on_enter': on_enter,
            'on_exit': on_exit
        }
        if self.current_state is None:
            self.current_state = state_name
    
    def add_transition(self, from_state, to_state, condition):
        """添加状态转移"""
        if from_state not in self.transitions:
            self.transitions[from_state] = []
        self.transitions[from_state].append({
            'to': to_state,
            'condition': condition
        })
    
    def update(self, **kwargs):
        """更新状态机"""
        if self.current_state in self.transitions:
            for trans in self.transitions[self.current_state]:
                if trans['condition'](**kwargs):
                    # 执行退出动作
                    if self.states[self.current_state]['on_exit']:
                        self.states[self.current_state]['on_exit']()
                    
                    # 切换状态
                    old_state = self.current_state
                    self.current_state = trans['to']
                    print(f"{self.name}: {old_state} → {self.current_state}")
                    
                    # 执行进入动作
                    if self.states[self.current_state]['on_enter']:
                        self.states[self.current_state]['on_enter']()
                    break

# 使用示例：智能台灯
def lamp_on():
    print("💡 台灯打开")

def lamp_off():
    print("🌑 台灯关闭")

def lamp_dim():
    print("🔅 台灯调暗")

# 创建状态机
lamp = StateMachine("智能台灯")

# 添加状态
lamp.add_state("OFF", on_enter=lamp_off)
lamp.add_state("ON", on_enter=lamp_on)
lamp.add_state("DIM", on_enter=lamp_dim)

# 添加转移
lamp.add_transition("OFF", "ON", lambda **kw: kw.get('touch', False))
lamp.add_transition("ON", "DIM", lambda **kw: kw.get('touch', False))
lamp.add_transition("DIM", "OFF", lambda **kw: kw.get('touch', False))
lamp.add_transition("DIM", "OFF", lambda **kw: kw.get('timeout', False))

# 模拟运行
import random
for _ in range(10):
    touch = random.random() > 0.6
    lamp.update(touch=touch)
```

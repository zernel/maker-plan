## 🧠 核心原理简述（给家长）

**C++** 是信息学奥赛的"官方语言"。NOI 系列赛事（CSP-J、CSP-S、NOIP、NOI）全部要求使用 C++。孩子之前学的 Python 是"脚本语言"，写起来方便但运行速度慢；C++ 是"编译语言"，代码先编译成机器码再运行，速度快很多。

为什么学 C++：
- **竞赛硬性要求**：NOI 信息学奥赛、CSP-J/S 认证只能用 C++
- **运行速度**：C++ 比 Python 快 10-100 倍，处理大数据量题目时是刚需
- **底层理解**：C++ 需要手动管理内存，能让孩子真正理解计算机怎么工作
- **思维升级**：C++ 的类型系统和编译检查能培养严谨的编程习惯

C++ 和 Python 的核心区别：

| 特性 | Python | C++ |
|------|--------|-----|
| 类型检查 | 运行时检查（动态类型） | 编译时检查（静态类型） |
| 内存管理 | 自动（垃圾回收） | 手动（new/delete） |
| 运行速度 | 较慢 | 很快 |
| 代码量 | 少 | 多（但更精确） |
| 适用场景 | AI、脚本、快速原型 | 竞赛、系统编程、游戏引擎 |

家长不必担心孩子"学不会"。C++ 的基础部分（变量、循环、函数）和 Python 逻辑一样，只是写法不同。类比一下：Python 像说白话文，C++ 像写文言文，表达的意思相同，只是格式更严格。

### 实验2: 📦 变量与数据类型

**零件清单**:
- 电脑（已安装 Dev-C++ 或 VS Code）
- 纸笔（记录变量类型和用途）

**玩法描述**:
1. 创建变量，对比 Python 和 C++ 的区别：
```cpp
#include <iostream>
#include <string>  // 使用字符串需要引入这个库
using namespace std;

int main() {
    // 整数类型
    int age = 10;  // int 存整数，类似 Python 的 int
    cout << "年龄：" << age << endl;

    // 浮点数类型
    double height = 1.45;  // double 存小数，类似 Python 的 float
    cout << "身高：" << height << "米" << endl;

    // 字符串类型
    string name = "小明";  // string 存文字，类似 Python 的 str
    cout << "姓名：" << name << endl;

    // 布尔类型
    bool is_student = true;  // bool 存真假，类似 Python 的 bool
    cout << "是学生吗？" << is_student << endl;  // 输出1表示true，0表示false

    // 常量（值不能改变）
    const int MAX_SCORE = 100;  // const 定义常量
    cout << "满分：" << MAX_SCORE << endl;

    return 0;
}
```

### 实验3: 🔁 循环与条件判断

**零件清单**:
- 电脑（已安装 Dev-C++ 或 VS Code）
- 纸笔（画流程图辅助理解）

**玩法描述**:
1. 用 C++ 实现循环，对比 Python 写法：
```cpp
#include <iostream>
using namespace std;

int main() {
    // for 循环：打印1到10
    cout << "=== for 循环 ===" << endl;
    for (int i = 1; i <= 10; i++) {  // i++ 等价于 i = i + 1
        cout << i << " ";
    }
    cout << endl;

    // while 循环：猜数字游戏
    cout << "=== while 循环 ===" << endl;
    int secret = 7;  // 秘密数字
    int guess = 0;   // 玩家猜测
    int attempts = 0; // 尝试次数

    while (guess != secret) {
        cout << "猜一个1-10的数字：";
        cin >> guess;  // 读取用户输入，类似 Python 的 input()
        attempts++;

        if (guess > secret) {
            cout << "太大了！" << endl;
        } else if (guess < secret) {
            cout << "太小了！" << endl;
        } else {
            cout << "恭喜你猜对了！用了" << attempts << "次" << endl;
        }
    }

    // 条件判断：成绩等级
    cout << "=== 条件判断 ===" << endl;
    int score;
    cout << "请输入成绩：";
    cin >> score;

    if (score >= 90) {
        cout << "优秀！" << endl;
    } else if (score >= 80) {
        cout << "良好！" << endl;
    } else if (score >= 60) {
        cout << "及格！" << endl;
    } else {
        cout << "需要加油！" << endl;
    }

    return 0;
}
```

### 实验4: 🧩 函数定义与调用

**零件清单**:

**玩法描述**:
1. 定义和调用函数，对比 Python 写法：
```cpp
#include <iostream>
#include <cmath>  // 数学库，提供 sqrt、pow 等函数
using namespace std;

// 函数定义：计算两个数的最大值
int max_num(int a, int b) {
    if (a > b) {
        return a;
    } else {
        return b;
    }
}

// 函数定义：计算阶乘
int factorial(int n) {
    if (n <= 1) {
        return 1;  // 递归终止条件
    }
    return n * factorial(n - 1);  // 递归调用
}

// 函数定义：判断是否为素数
bool is_prime(int n) {
    if (n < 2) return false;
    for (int i = 2; i * i <= n; i++) {  // 只需检查到 sqrt(n)
        if (n % i == 0) {
            return false;
        }
    }
    return true;
}

int main() {
    // 调用函数
    cout << "最大值：" << max_num(10, 20) << endl;
    cout << "5的阶乘：" << factorial(5) << endl;

    // 找出1-50之间的所有素数
    cout << "1-50的素数：";
    for (int i = 2; i <= 50; i++) {
        if (is_prime(i)) {
            cout << i << " ";
        }
    }
    cout << endl;

    return 0;
}
```

```python
# Python 版本（对比）
def max_num(a, b):

        return a
    else:
        return b

def factorial(n):

        return 1
    return n * factorial(n - 1)

def is_prime(n):

        return False
    for i in range(2, int(n**0.5) + 1):
```

### 实验5: 🏗️ 结构体与类

**零件清单**:
- 电脑（已安装 Dev-C++ 或 VS Code）
- 纸笔（画类的属性和方法关系图）

**玩法描述**:
1. 用 C++ 的结构体和类来管理学生信息，对比 Python：
```cpp
#include <iostream>
#include <string>
#include <vector>  // 动态数组，类似 Python 的 list
using namespace std;

// 结构体：简单数据集合
struct Student {
    string name;
    int age;
    double score;
};

// 类：包含数据和方法
class GradeManager {
private:  // 私有成员，外部不能直接访问
    vector<Student> students;  // 存储所有学生

public:  // 公有成员，外部可以访问
    // 添加学生
    void add_student(string name, int age, double score) {
        Student s;
        s.name = name;
        s.age = age;
        s.score = score;
        students.push_back(s);  // 添加到列表末尾
        cout << "已添加学生：" << name << endl;
    }

    // 计算平均分
    double get_average() {
        if (students.empty()) return 0;
        double total = 0;
        for (int i = 0; i < students.size(); i++) {
            total += students[i].score;
        }
        return total / students.size();
    }

    // 找最高分
    Student get_top_student() {
        Student top = students[0];
        for (int i = 1; i < students.size(); i++) {
            if (students[i].score > top.score) {
                top = students[i];
            }
        }
        return top;
    }

    // 显示所有学生
    void show_all() {
        cout << "=== 学生列表 ===" << endl;
        for (int i = 0; i < students.size(); i++) {
            cout << students[i].name << "，"
                 << students[i].age << "岁，"
                 << "成绩：" << students[i].score << endl;
        }
        cout << "平均分：" << get_average() << endl;
    }
};

int main() {
    // 使用结构体
    Student s1 = {"小明", 10, 95.5};
    cout << s1.name << "的成绩：" << s1.score << endl;

    // 使用类
    GradeManager manager;
    manager.add_student("小明", 10, 95.5);
    manager.add_student("小红", 11, 98.0);
    manager.add_student("小刚", 10, 88.5);
    manager.show_all();

    Student top = manager.get_top_student();
    cout << "最高分：" << top.name << " " << top.score << endl;

    return 0;
}
```

```python
# Python 版本（对比）
class Student:
    def __init__(self, name, age, score):
        self.name = name
        self.age = age
        self.score = score

class GradeManager:
    def __init__(self):
        self.students = []

    def add_student(self, name, age, score):
        s = Student(name, age, score)
        self.students.append(s)
        print(f"已添加学生：{name}")

    def get_average(self):
        if not self.students:
            return 0
        return sum(s.score for s in self.students) / len(self.students)
```

## 📝 家长小贴士

> 💡 **关键点**: C++ 的入门门槛比 Python 高，主要因为语法更严格（必须写分号、必须声明类型）。但基础概念（变量、循环、函数）和 Python 是一样的。让孩子先在纸上画出逻辑，再翻译成 C++ 代码，会更容易上手。

> ⚠️ **常见误区**: 不要一上来就教指针和内存管理。对于 10-12 岁的孩子，先掌握变量、循环、函数、结构体这四个核心概念就够了。指针是进阶内容，留到竞赛冲刺阶段再学。

> 🛠️ **工具选择**: Windows 推荐 Dev-C++（安装简单，界面简洁），Mac/Linux 推荐 VS Code + g++ 编译器。不要用 Visual Studio（太重了），也不要用在线编译器（不方便保存代码和调试）。

# 🧒 Track B: 7岁小学二年级路径

> **适合年龄**: 7-8岁小学低年级
> **学习周期**: 6-12个月（每周2-3次，每次30-45分钟）
> **核心理念**: 从实体编程过渡到屏幕编程，激发对电子世界的兴趣

---

## 🎯 核心目标

| 维度 | 目标 |
|------|------|
| 💻 **编程能力** | 掌握图形化编程基础（MakeCode） |
| 🔌 **硬件认知** | 认识传感器、LED、电机等电子元件 |
| 🧠 **计算思维** | 理解变量、函数、事件驱动等概念 |
| 🛠️ **动手能力** | 完成简单电路连接和小车制作 |

---

## 🛠️ 推荐器材

### 主力器材
- **Micro:bit V2** - 英国BBC出品的教育微型电脑，内置多种传感器
- **Micro:bit 入门套件** - 包含面包板、杜邦线、LED、电阻等基础元件

### 扩展套件
- 超声波传感器套件
- NeoPixel LED灯带
- 小车底盘 + 电机驱动板
- 红外循迹传感器

## 🎮 教学方法

### 图形化编程入门
使用 MakeCode 图形化编程环境，通过拖拽积木块来编写程序。

参考: [[02-Concepts/实体计算]]

### 项目驱动学习
每个节点都有明确的项目目标，孩子在完成项目的过程中学习。

参考: [[02-Concepts/机器人教育]]

### 渐进式难度
从内置传感器开始，逐步过渡到外接传感器和电机控制。

## ⏰ 时间安排建议

| 阶段 | 时长 | 内容 |

## 📝 家长提示

1. **屏幕时间管理**: MakeCode编程是"创造性屏幕时间"，和玩游戏不同
2. **鼓励探索**: 让孩子自己尝试修改参数，观察效果
3. **记录作品**: 拍照或录视频，记录孩子的成长
4. **安全第一**: 连接电路时注意正负极，避免短路

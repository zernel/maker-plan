## 🧠 核心原理简述（给家长）

**红外循迹传感器**能检测地面颜色，通常用于检测黑线。

技术细节：
- 原理: 红外发射管发出红外光，接收管检测反射光
- 输出: 数字信号（0或1）
- 检测: 黑色吸收红外光（输出1），白色反射红外光（输出0）
- 安装: 通常安装2-3个，位于车底

循迹算法：
1. 左传感器检测到黑线 → 左转
2. 右传感器检测到黑线 → 右转
3. 两个传感器都没检测到 → 直行
4. 两个传感器都检测到 → 停止或特殊处理

孩子将学习：
- **传感器应用**: 红外传感器的实际使用
- **反馈控制**: 根据传感器反馈调整行为
- **算法思维**: 设计简单的决策逻辑

## 🔬 趣味实验清单

### 实验1: ⚫ 黑线跟踪

**零件清单**:
- 组装好的小车（参考B07）
- 红外循迹传感器（2个）
- 黑色电工胶带
- 白色地板或大白纸
- 杜邦线若干

**玩法描述**:
1. 安装循迹传感器：
   - 两个传感器分别安装在车底左右两侧
   - 间距略大于黑线宽度
2. 连接电路：
   - 左传感器 → P0
   - 右传感器 → P1
3. 在地面用黑色胶带贴一条直线
4. 编写循迹程序：
   - 左传感器检测到黑线 → 左转
   - 右传感器检测到黑线 → 右转
   - 都没检测到 → 直行
5. 下载程序，测试循迹

**观察与思考**:
- 小车能稳定跟踪黑线吗？
- 速度太快会怎样？
- 传感器离地面多高最合适？

---

### 实验2: 🏁 简单赛道

**零件清单**:
- 循迹小车
- 黑色电工胶带
- 大白纸或地板
- 计时器

**玩法描述**:
1. 设计一个简单赛道：
   - 直线段
   - 90度弯道
   - 圆形弯道
2. 用黑色胶带在地面贴出赛道
3. 编写更精细的循迹算法
4. 测试小车能否完整跑完赛道
5. 计时，挑战最快圈速

**观察与思考**:
- 直线段和弯道，哪个更难？
- 小车在弯道为什么会"抖动"？
- 如何让弯道更顺畅？

---

### 实验3: 🎮 循迹+避障组合

**零件清单**:
- 循迹小车
- 红外循迹传感器
- 超声波传感器（可选）
- 赛道和障碍物

**玩法描述**:
1. 在赛道上放置障碍物
2. 编写组合程序：
   - 正常循迹
   - 检测到障碍物 → 停车
   - 障碍物移开 → 继续循迹
3. 测试组合功能
4. 尝试设计更复杂的赛道

**观察与思考**:
- 循迹和避障，哪个优先级更高？
- 如何让小车"记住"赛道？
- 这和真实的自动驾驶有什么相似之处？

## 📝 家长小贴士

> 💡 **关键点**: 循迹小车是"感知-决策-执行"循环的完美示例。传感器感知环境，程序做出决策，电机执行动作。这是所有自主系统的基础。

> ⚠️ **常见误区**: 循迹调试需要耐心。小车不会一次就完美运行，需要反复调整传感器位置、程序参数。这个调试过程本身就是宝贵的学习。

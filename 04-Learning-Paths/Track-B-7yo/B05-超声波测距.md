## 🧠 核心原理简述（给家长）

**超声波传感器**能测量物体距离，就像蝙蝠的"回声定位"。

技术细节：
- 原理: 发出超声波脉冲，测量回波时间
- 测量范围: 2cm - 400cm
- 精度: ±3mm
- 常见型号: HC-SR04

工作流程：
1. Trig引脚发出超声波脉冲
2. 超声波碰到物体反射回来
3. Echo引脚接收回波
4. 根据时间差计算距离

孩子将学习：
- **电路连接**: 4根线的连接（VCC、Trig、Echo、GND）
- **物理原理**: 声速与距离的关系
- **数据处理**: 将时间转化为距离

## 🔬 趣味实验清单

### 实验1: 📊 距离显示器

**零件清单**:
- Micro:bit V2 一块
- 超声波传感器（HC-SR04）
- 面包板
- 杜邦线4根
- USB数据线

**玩法描述**:
1. 连接电路：
   - VCC → 3V
   - GND → GND
   - Trig → P0
   - Echo → P1
2. 在MakeCode中添加超声波扩展
3. 使用"超声波距离"积木块
4. 在LED上显示距离（厘米）
5. 用手在传感器前移动，观察数值变化
6. 测量不同物体的距离

**观察与思考**:
- 最远能测多远？
- 数值稳定吗？有没有跳动？
- 什么材质的物体测距更准确？

---

### 实验2: 📐 电子尺子

**零件清单**:
- Micro:bit V2 一块
- 超声波传感器
- 面包板和杜邦线
- 真实尺子（用于对比）
- 不同大小的物体

**玩法描述**:
1. 用超声波测量一本书的长度
2. 用真实尺子测量同一本书
3. 比较两个数值，讨论误差
4. 测量桌子宽度
5. 测量房间宽度
6. 记录数据，讨论精度

**观察与思考**:
- 超声波测量和尺子测量，哪个更准？
- 为什么会有误差？
- 什么情况下超声波测距更好用？

---

### 实验3: 🚧 距离报警器

**零件清单**:
- Micro:bit V2 一块
- 超声波传感器
- 面包板和杜邦线
- 蜂鸣器（可选）

**玩法描述**:
1. 编写条件判断：
   - 如果距离 < 10cm，显示"太近！"并发出警报
   - 如果距离 > 100cm，显示"太远！"
   - 否则显示"安全"
2. 下载程序
3. 用手靠近传感器，测试报警
4. 调整报警距离阈值

**观察与思考**:
- 多近算"太近"？
- 这个技术可以用在哪些场景？（倒车雷达、自动门）
- 如何让报警更灵敏？

## 📝 家长小贴士

> 💡 **关键点**: 超声波传感器是孩子第一次接触"物理世界数字化"。它把"距离"这个物理量转化为"数字"，这种抽象能力是编程思维的核心。

> ⚠️ **常见误区**: 超声波传感器对软质物体（如棉花）和倾斜表面的测量可能不准确。这是正常现象，可以借此讨论"传感器的局限性"。

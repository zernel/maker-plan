## 🧠 核心原理简述（给家长）

**物联网 (IoT)** 是将物理设备连接到互联网的技术。ESP32是低成本、高性能的物联网芯片。

ESP32特性：
- **WiFi**: 连接家庭无线网络
- **蓝牙**: 低功耗蓝牙通信
- **GPIO**: 丰富的输入输出引脚
- **MicroPython支持**: 可用Python编程

物联网架构：
1. **感知层**: 传感器采集数据
2. **网络层**: WiFi传输数据
3. **应用层**: 云端存储和展示

孩子将学习：
- **WiFi连接**: 让设备联网
- **HTTP通信**: 与云端服务器交互
- **数据上传**: 将传感器数据上传到云端

## 🔬 趣味实验清单

### 实验1: 📶 连接WiFi

**零件清单**:
- ESP32开发板一块
- USB数据线一根
- 电脑（安装Thonny）
- WiFi网络

**玩法描述**:
1. 安装Thonny，配置ESP32固件
2. 连接ESP32到电脑
3. 编写WiFi连接程序：
```python
import network
import time

# WiFi配置
SSID = "你的WiFi名称"
PASSWORD = "你的WiFi密码"

# 连接WiFi
wlan = network.WLAN(network.STA_IF)
wlan.active(True)
wlan.connect(SSID, PASSWORD)

# 等待连接
while not wlan.isconnected():
    time.sleep(1)
    print("连接中...")

print("已连接!")
print("IP地址:", wlan.ifconfig()[0])
```
4. 下载程序，观察连接过程
5. 记录分配的IP地址

**观察与思考**:
- WiFi连接需要什么信息？
- IP地址是什么？有什么用？
- 如果WiFi密码错误会怎样？

**⚠️ WiFi安全提示**:
- 不要在代码中硬编码真实WiFi密码（分享代码时会泄露）
- 学习使用配置文件或环境变量存储敏感信息
- 只在安全的网络环境下测试物联网设备
- 不要将物联网设备暴露在公网（使用内网测试）

### 实验2: 💡 LED远程控制

**零件清单**:
- ESP32开发板一块
- LED一个
- 电阻（220欧姆）
- 面包板
- 杜邦线
- 电脑

**玩法描述**:
1. 连接LED到ESP32的GPIO2
2. 编写Web服务器程序：
```python
import network
import socket
from machine import Pin
import time

# WiFi连接
SSID = "你的WiFi名称"
PASSWORD = "你的WiFi密码"

wlan = network.WLAN(network.STA_IF)
wlan.active(True)
wlan.connect(SSID, PASSWORD)

while not wlan.isconnected():
    time.sleep(1)

print("IP:", wlan.ifconfig()[0])

# LED设置
led = Pin(2, Pin.OUT)

# HTML页面
html = """<!DOCTYPE html>
<html>
<head><title>LED Control</title></head>
<body>
<h1>LED Control</h1>
<a href="/on"><button>ON</button></a>
<a href="/off"><button>OFF</button></a>
</body>
</html>
"""

# Web服务器
addr = socket.getaddrinfo('0.0.0.0', 80)[0][-1]
s = socket.socket()
s.bind(addr)
s.listen(1)

while True:
    cl, addr = s.accept()
    request = cl.recv(1024)
    request = str(request)
    
    if '/on' in request:
        led.value(1)
    elif '/off' in request:
        led.value(0)
    
    cl.send('HTTP/1.0 200 OK\r\nContent-type: text/html\r\n\r\n')
    cl.send(html)
    cl.close()
```
3. 下载程序，用手机浏览器访问IP地址
4. 点击按钮控制LED

**观察与思考**:
- 手机和ESP32是如何通信的？
- HTML是什么？它的作用是什么？
- 如何让界面更美观？

### 实验3: 📊 数据可视化

**零件清单**:
- ESP32开发板一块
- DHT11温湿度传感器
- 面包板和杜邦线
- 电脑

**玩法描述**:
1. 连接DHT11传感器
2. 编写数据上传程序：
```python
import network
import socket
import dht
import machine
import time
import json

# WiFi连接
SSID = "你的WiFi名称"
PASSWORD = "你的WiFi密码"

wlan = network.WLAN(network.STA_IF)
wlan.active(True)
wlan.connect(SSID, PASSWORD)

while not wlan.isconnected():
    time.sleep(1)

# 传感器
sensor = dht.DHT11(machine.Pin(4))

# Web服务器
addr = socket.getaddrinfo('0.0.0.0', 80)[0][-1]
s = socket.socket()
s.bind(addr)
s.listen(1)

while True:
    sensor.measure()
    temp = sensor.temperature()
    humi = sensor.humidity()
    
    cl, addr = s.accept()
    request = cl.recv(1024)
    
    html = f"""<!DOCTYPE html>
<html>
<head><title>Weather Station</title></head>
<body>
<h1>Weather Station</h1>
<p>Temperature: {temp}°C</p>
<p>Humidity: {humi}%</p>
<meta http-equiv="refresh" content="5">
</body>
</html>
"""
    
    cl.send('HTTP/1.0 200 OK\r\nContent-type: text/html\r\n\r\n')
    cl.send(html)
    cl.close()
    
    time.sleep(5)
```
3. 用浏览器查看实时数据
4. 观察数据变化

**观察与思考**:
- 数据多久更新一次？
- 如何让数据更美观？（图表、颜色）
- 如何将数据保存到云端？

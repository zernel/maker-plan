## 🧠 核心原理简述（给家长）

**传感器读取**是将物理世界的信息转化为数字数据的过程。MicroPython提供了丰富的API来读取传感器。

技术细节：
- **模拟输入**: 读取0-1023的连续值（如光线、温度）
- **数字输入**: 读取0或1的离散值（如按钮、循迹）
- **I2C通信**: 多设备共用两根线通信
- **数据处理**: 滤波、校准、单位转换

孩子将学习：
- **数据采集**: 如何从传感器获取原始数据
- **数据处理**: 如何将原始数据转化为有意义的数值
- **数据分析**: 如何根据数据做出判断

## 🔬 趣味实验清单

### 实验1: 🌡️ 温度监控

**零件清单**:
- Micro:bit V2 一块
- USB数据线一根
- 电脑

**玩法描述**:
1. 读取内置温度传感器：
```python
from microbit import *

while True:
    temp = temperature()
    display.scroll(str(temp) + "C")
    sleep(2000)
```
2. 添加温度报警：
```python
from microbit import *

while True:
    temp = temperature()
    if temp > 30:
        display.scroll("HOT!")
    elif temp < 15:
        display.scroll("COLD")
    else:
        display.scroll(str(temp))
    sleep(1000)
```
3. 记录温度变化：
```python
from microbit import *

temps = []
for i in range(10):
    temps.append(temperature())
    sleep(1000)

display.scroll("Avg:" + str(sum(temps)//len(temps)))
```

**观察与思考**:
- 温度读数准确吗？和温度计对比一下
- 如何让读数更稳定？（多次测量取平均）
- 温度变化有什么规律？

### 实验2: 💡 光线检测

**零件清单**:
- Micro:bit V2 一块
- USB数据线一根
- 电脑
- 外接光敏电阻（可选）
- 面包板和电阻（可选）

**玩法描述**:
1. 读取内置光线传感器：
```python
from microbit import *

while True:
    light = display.read_light_level()
    display.scroll(str(light))
    sleep(500)
```
2. 光线条形图：
```python
from microbit import *

while True:
    light = display.read_light_level()
    level = light * 5 // 255
    for x in range(5):
        for y in range(5):
            if y >= 5 - level:
                display.set_pixel(x, y, 9)
            else:
                display.set_pixel(x, y, 0)
    sleep(100)
```
3. 自动亮度调节：
```python
from microbit import *

while True:
    light = display.read_light_level()
    if light < 50:
        display.set_brightness(9)
    else:
        display.set_brightness(3)
    display.scroll("Light:" + str(light))
    sleep(1000)
```

**观察与思考**:
- 光线强度范围是多少？
- 什么环境最亮？什么环境最暗？
- 如何用光线控制其他设备？

### 实验3: 📏 距离测量

**零件清单**:
- Micro:bit V2 一块
- 超声波传感器（HC-SR04）
- 面包板
- 杜邦线4根
- 电脑

**玩法描述**:
1. 连接超声波传感器：
   - VCC → 3V
   - GND → GND
   - Trig → P0
   - Echo → P1
2. 编写测距程序：
```python
from microbit import *

def get_distance():
    pin0.write_digital(1)
    sleep(0.01)
    pin0.write_digital(0)
    duration = pin1.read_pulse(1)
    distance = duration * 0.034 / 2
    return distance

while True:
    dist = get_distance()
    display.scroll(str(int(dist)) + "cm")
    sleep(500)
```
3. 距离报警器：
```python
from microbit import *

while True:
    dist = get_distance()
    if dist < 10:
        display.scroll("TOO CLOSE!")
        pin2.write_digital(1)  # 蜂鸣器
    else:
        display.scroll(str(int(dist)))
        pin2.write_digital(0)
    sleep(200)
```

**观察与思考**:
- 测量精度如何？和尺子对比
- 什么物体会干扰测量？
- 如何提高测量稳定性？

## 📝 家长小贴士

> 💡 **关键点**: 传感器读取是"物理世界数字化"的核心。孩子通过代码将物理量转化为数字，这是理解物联网、大数据的基础。

> ⚠️ **常见误区**: 传感器数据可能有噪声。教孩子理解"数据不完美"，学习用滤波算法（如移动平均）提高数据质量。

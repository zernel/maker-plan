## 🧠 核心原理简述（给家长）

**循迹算法**从简单的开关控制到复杂的PID控制，体现了控制理论的进阶。

算法层次：
1. **开关控制**: 检测到线就转，否则直行（简单但抖动）
2. **比例控制 (P)**: 偏差越大，转向越大（更平滑）
3. **积分控制 (I)**: 消除累积误差
4. **微分控制 (D)**: 预测未来趋势，减少超调
5. **PID控制**: 综合P、I、D，实现最优控制

孩子将理解：
- **反馈控制**: 根据误差调整输出
- **参数调优**: 如何找到最佳PID参数
- **系统稳定性**: 避免振荡和失控

## 🔬 趣味实验清单

### 实验1: ⚫ 黑线跟踪

**零件清单**:
- Maqueen小车套件（带循迹传感器）
- Micro:bit V2 一块
- 黑色电工胶带
- 白色地板或大白纸
- 电池

**玩法描述**:
1. 检查循迹传感器：
   - Maqueen底部有3个循迹传感器（左、中、右）
   - 黑色输出1，白色输出0
2. 编写基础循迹程序：
```python
from microbit import *
import maqueen

while True:
    # 读取循迹传感器
    left = maqueen.read_patrol(maqueen.Patrol.PATROL_LEFT)
    right = maqueen.read_patrol(maqueen.Patrol.PATROL_RIGHT)
    
    if left == 1 and right == 1:  # 两个都在黑线上
        maqueen.motor_run(maqueen.Motors.ALL, maqueen.Dir.CW, 80)
    elif left == 1:  # 左侧检测到线
        maqueen.motor_run(maqueen.Motors.M1, maqueen.Dir.CW, 0)
        maqueen.motor_run(maqueen.Motors.M2, maqueen.Dir.CW, 80)
    elif right == 1:  # 右侧检测到线
        maqueen.motor_run(maqueen.Motors.M1, maqueen.Dir.CW, 80)
        maqueen.motor_run(maqueen.Motors.M2, maqueen.Dir.CW, 0)
    else:  # 都没检测到，继续前进
        maqueen.motor_run(maqueen.Motors.ALL, maqueen.Dir.CW, 60)
    
    sleep(50)
```
3. 在直线赛道上测试
4. 观察小车能否稳定跟踪

**观察与思考**:
- 小车走直线稳定吗？会不会左右摆动？
- 速度对循迹有什么影响？
- 传感器离黑线多远检测不到？

### 实验2: 🔄 弯道处理

**零件清单**:
- 同实验1
- 弯道赛道

**玩法描述**:
1. 设计弯道赛道（用黑色胶带）
2. 改进循迹算法，支持弯道：
```python
from microbit import *
import maqueen

# 速度参数
BASE_SPEED = 60
TURN_SPEED = 40

while True:
    left = maqueen.read_patrol(maqueen.Patrol.PATROL_LEFT)
    right = maqueen.read_patrol(maqueen.Patrol.PATROL_RIGHT)
    
    if left == 0 and right == 0:  # 两个都在白色上
        maqueen.motor_run(maqueen.Motors.ALL, maqueen.Dir.CW, BASE_SPEED)
    elif left == 1 and right == 1:  # 两个都在黑线上
        maqueen.motor_run(maqueen.Motors.ALL, maqueen.Dir.CW, BASE_SPEED)
    elif left == 1:  # 左转
        maqueen.motor_run(maqueen.Motors.M1, maqueen.Dir.CW, TURN_SPEED)
        maqueen.motor_run(maqueen.Motors.M2, maqueen.Dir.CW, BASE_SPEED)
    elif right == 1:  # 右转
        maqueen.motor_run(maqueen.Motors.M1, maqueen.Dir.CW, BASE_SPEED)
        maqueen.motor_run(maqueen.Motors.M2, maqueen.Dir.CW, TURN_SPEED)
    
    sleep(30)
```
3. 测试弯道循迹
4. 调整速度参数

**观察与思考**:
- 直道和弯道应该用不同速度吗？
- 转弯太急会怎样？转弯太慢会怎样？
- 如何让弯道更顺畅？

### 实验3: 🏎️ 速度优化

**零件清单**:
- 同实验1
- 计时器

**玩法描述**:
1. 实现PID循迹算法：
```python
from microbit import *
import maqueen

# PID参数
Kp = 0.5
Ki = 0.01
Kd = 0.3

# 状态变量
last_error = 0
integral = 0

def get_error():
    left = maqueen.read_patrol(maqueen.Patrol.PATROL_LEFT)
    right = maqueen.read_patrol(maqueen.Patrol.PATROL_RIGHT)
    
    if left == 1 and right == 1:
        return 0
    elif left == 1:
        return -1
    elif right == 1:
        return 1
    else:
        return last_error  # 保持上次状态

while True:
    error = get_error()
    
    # PID计算
    integral += error
    derivative = error - last_error
    
    correction = Kp * error + Ki * integral + Kd * derivative
    
    # 电机控制
    left_speed = int(80 + correction * 30)
    right_speed = int(80 - correction * 30)
    
    maqueen.motor_run(maqueen.Motors.M1, maqueen.Dir.CW, max(0, min(100, left_speed)))
    maqueen.motor_run(maqueen.Motors.M2, maqueen.Dir.CW, max(0, min(100, right_speed)))
    
    last_error = error
    sleep(20)
```
2. 测试PID循迹
3. 记录完成赛道的时间
4. 调整PID参数，优化速度

**观察与思考**:
- PID参数对循迹有什么影响？
- Kp太大的结果是什么？太小呢？
- 如何找到最佳PID参数？

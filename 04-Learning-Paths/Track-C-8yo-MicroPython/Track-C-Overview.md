# 🧒 Track C: 8岁MicroPython进阶路径

> **适合年龄**: 8-10岁小学中高年级
> **学习周期**: 12-18个月（每周2-3次，每次45-60分钟）
> **核心理念**: 从图形化编程过渡到文本编程，培养工程思维和创新能力

---

## 🎯 核心目标

| 维度 | 目标 |
|------|------|
| 💻 **编程能力** | 掌握MicroPython文本编程 |
| 🔌 **硬件能力** | 独立完成传感器连接和调试 |
| 🧠 **算法思维** | 理解PID控制、状态机等算法 |
| 🛠️ **工程能力** | 完成综合项目（小车、物联网） |
| 🏆 **竞赛准备** | 为白名单竞赛打基础 |

### 从图形化到文本
使用Mu Editor或Thonny作为MicroPython编程环境，逐步从MakeCode过渡。

参考: [[02-Concepts/计算思维]]

### 项目驱动学习
每个节点都有明确的项目目标，从简单到复杂逐步进阶。

参考: [[02-Concepts/机器人教育]]

### 竞赛导向
学习内容与白名单竞赛（如NOC、蓝桥杯）对接，为参赛做准备。

参考: [[01-White-List-Competitions/]]

## 📝 家长提示

1. **代码调试是学习**: 报错是正常的，调试过程比结果更重要
2. **鼓励独立思考**: 让孩子先自己尝试，而不是直接给答案
3. **记录代码**: 用Git或文件夹管理代码，记录成长轨迹
4. **安全意识**: 电路连接注意正负极，避免短路和过热

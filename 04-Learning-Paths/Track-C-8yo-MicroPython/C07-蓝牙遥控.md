## 🧠 核心原理简述（给家长）

**蓝牙通信**是短距离无线通信技术，广泛用于手机、耳机、智能家居。

技术细节：
- **蓝牙协议**: 2.4GHz频段，通信距离约10米
- **BLE (低功耗蓝牙)**: Micro:bit使用BLE，功耗低
- **服务和特征**: BLE通信的基本单元
- **配对**: 设备之间建立连接的过程

孩子将学习：
- **无线通信**: 理解数据如何无线传输
- **协议设计**: 设计简单的通信协议
- **人机交互**: 设计遥控界面

## 🔬 趣味实验清单

### 实验1: 📱 手机遥控

**零件清单**:
- Maqueen小车套件
- Micro:bit V2 一块
- 手机（安装Micro:bit BlueAPP）
- 电池

**玩法描述**:
1. 下载Micro:bit BlueAPP（iOS/Android）
2. 编写蓝牙遥控程序：
```python
from microbit import *
import maqueen
import bluetooth

# 蓝牙服务
uart.init(baudrate=9600)

while True:
    if uart.any():
        cmd = uart.read().decode()
        
        if cmd == "F":  # 前进
            maqueen.motor_run(maqueen.Motors.ALL, maqueen.Dir.CW, 80)
        elif cmd == "B":  # 后退
            maqueen.motor_run(maqueen.Motors.ALL, maqueen.Dir.CCW, 80)
        elif cmd == "L":  # 左转
            maqueen.motor_run(maqueen.Motors.M1, maqueen.Dir.CW, 0)
            maqueen.motor_run(maqueen.Motors.M2, maqueen.Dir.CW, 80)
        elif cmd == "R":  # 右转
            maqueen.motor_run(maqueen.Motors.M1, maqueen.Dir.CW, 80)
            maqueen.motor_run(maqueen.Motors.M2, maqueen.Dir.CW, 0)
        elif cmd == "S":  # 停止
            maqueen.motor_stop(maqueen.Motors.ALL)
    
    sleep(50)
```
3. 用手机APP连接Micro:bit
4. 发送指令控制小车

**观察与思考**:
- 蓝牙连接稳定吗？有延迟吗？
- 通信距离有多远？
- 如何设计更直观的遥控界面？

### 实验2: 📡 双Micro:bit通信

**零件清单**:
- Micro:bit V2 两块
- Maqueen小车套件
- 电池

**玩法描述**:
1. 遥控器端程序：
```python
from microbit import *
import radio

radio.on()
radio.config(channel=7)

while True:
    if button_a.is_pressed():
        radio.send("F")
        sleep(200)
    elif button_b.is_pressed():
        radio.send("B")
        sleep(200)
    elif accelerometer.was_gesture("shake"):
        radio.send("S")
        sleep(200)
    elif accelerometer.was_gesture("left"):
        radio.send("L")
        sleep(200)
    elif accelerometer.was_gesture("right"):
        radio.send("R")
        sleep(200)
    
    sleep(50)
```
2. 小车端程序：
```python
from microbit import *
import maqueen
import radio

radio.on()
radio.config(channel=7)

while True:
    msg = radio.receive()
    
    if msg:
        if msg == "F":
            maqueen.motor_run(maqueen.Motors.ALL, maqueen.Dir.CW, 80)
        elif msg == "B":
            maqueen.motor_run(maqueen.Motors.ALL, maqueen.Dir.CCW, 80)
        elif msg == "L":
            maqueen.motor_run(maqueen.Motors.M1, maqueen.Dir.CW, 0)
            maqueen.motor_run(maqueen.Motors.M2, maqueen.Dir.CW, 80)
        elif msg == "R":
            maqueen.motor_run(maqueen.Motors.M1, maqueen.Dir.CW, 80)
            maqueen.motor_run(maqueen.Motors.M2, maqueen.Dir.CW, 0)
        elif msg == "S":
            maqueen.motor_stop(maqueen.Motors.ALL)
    
    sleep(50)
```
3. 测试双机遥控

**观察与思考**:
- 蓝牙和无线电有什么区别？
- 通信距离有多远？
- 如何避免信号干扰？

### 实验3: 🎮 体感遥控

**零件清单**:
- Micro:bit V2 两块
- Maqueen小车套件
- 电池

**玩法描述**:
1. 遥控器端程序（体感控制）：
```python
from microbit import *
import radio

radio.on()
radio.config(channel=7)

while True:
    x = accelerometer.get_x()
    y = accelerometer.get_y()
    
    # 根据倾斜方向发送指令
    if y < -300:  # 前倾
        radio.send("F")
    elif y > 300:  # 后倾
        radio.send("B")
    elif x < -300:  # 左倾
        radio.send("L")
    elif x > 300:  # 右倾
        radio.send("R")
    else:  # 平放
        radio.send("S")
    
    sleep(100)
```
2. 小车端接收并执行
3. 测试体感控制

**观察与思考**:
- 体感控制比按钮控制更直观吗？
- 倾斜角度和速度应该有关系吗？
- 如何让控制更精准？

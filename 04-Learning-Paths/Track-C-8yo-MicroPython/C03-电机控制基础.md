## 🧠 核心原理简述（给家长）

**电机控制**涉及电力电子和自动控制的基础知识。

核心技术：
- **PWM (脉宽调制)**: 通过改变脉冲宽度控制电机速度
- **H桥电路**: 控制电机正反转的电路
- **电机驱动板**: 放大Micro:bit信号，驱动大电流电机
- **编码器**: 测量电机转速的传感器

孩子将学习：
- **PWM原理**: 用代码控制电机速度
- **正反转控制**: 改变电流方向
- **速度闭环**: 用编码器反馈控制速度

## 🔬 趣味实验清单

### 实验1: 🔄 调速控制

**零件清单**:
- Micro:bit V2 一块
- 直流电机一个
- 电机驱动板（L298N或类似）
- 电池盒 + 4节AA电池
- 杜邦线若干

**玩法描述**:
1. 连接电路：
   - 电机驱动板IN1 → P0
   - 电机驱动板IN2 → P1
   - 电机驱动板ENA → P2（PWM）
   - 电池盒 → 驱动板电源输入
2. 编写调速程序：
```python
from microbit import *

# 设置引脚
pin0.write_digital(0)  # IN1
pin1.write_digital(1)  # IN2

# PWM调速
for speed in range(0, 1023, 100):
    pin2.write_analog(speed)
    display.scroll(str(speed))
    sleep(1000)
```
3. 测试不同速度
4. 尝试用按钮控制速度

**观察与思考**:
- PWM值和电机速度是什么关系？
- 最低多少PWM值电机才会转？
- PWM值太高会怎样？

### 实验2: ↔️ 正反转控制

**零件清单**:
- 同实验1

**玩法描述**:
1. 编写正反转程序：
```python
from microbit import *

while True:
    # 正转
    pin0.write_digital(0)
    pin1.write_digital(1)
    pin2.write_analog(512)
    display.scroll("FWD")
    sleep(2000)
    
    # 停止
    pin2.write_analog(0)
    display.scroll("STOP")
    sleep(1000)
    
    # 反转
    pin0.write_digital(1)
    pin1.write_digital(0)
    pin2.write_analog(512)
    display.scroll("REV")
    sleep(2000)
```
2. 测试正反转
3. 用按钮控制方向

**观察与思考**:
- IN1和IN2的电平组合有几种？
- 00、01、10、11分别对应什么状态？
- 如何实现"刹车"？

### 实验3: 🌉 H桥原理

**零件清单**:
- 电机驱动板
- 电机
- LED（用于可视化）
- 面包板

**玩法描述**:
1. 用LED演示H桥：
   - 4个LED代表H桥的4个开关
   - 不同开关组合，LED亮灭不同
2. 讲解H桥原理：
   - S1+S4闭合 → 正转
   - S2+S3闭合 → 反转
   - 全部断开 → 自由转动
   - S1+S3或S2+S4闭合 → 制动
3. 用代码模拟H桥行为
4. 讨论：为什么需要H桥？

**观察与思考**:
- H桥为什么叫"H"桥？
- 直接用Micro:bit引脚驱动电机会怎样？
- 电机驱动板的作用是什么？

## 📝 家长小贴士

> 💡 **关键点**: PWM是控制电机速度的关键技术。它不是真正改变电压，而是快速开关电源，通过"占空比"来控制平均功率。

> ⚠️ **常见误区**: 电机是感性负载，断开瞬间会产生高压脉冲。电机驱动板的续流二极管就是为了保护电路。

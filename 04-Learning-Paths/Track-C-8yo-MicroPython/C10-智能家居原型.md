## 🧠 核心原理简述（给家长）

**智能家居**通过传感器和控制器，实现家居设备的自动化控制。

核心技术：
- **继电器**: 用弱电控制强电（如控制220V灯具）
- **传感器网络**: 多个传感器协同工作
- **自动化逻辑**: 根据条件自动执行动作
- **远程控制**: 通过手机APP或网页控制

安全注意事项：
- 强电操作必须在家长指导下进行
- 继电器只能控制低功率设备
- 不要直接接触220V电源

## 🔬 趣味实验清单

### 实验1: 💡 自动灯光

**零件清单**:
- ESP32开发板一块
- 继电器模块一个
- LED灯带（12V）
- 12V电源适配器
- 光敏电阻
- 电阻（10K欧姆）
- 面包板
- 杜邦线若干

**玩法描述**:
1. 连接电路：
   - 继电器信号输入 → GPIO2
   - 光敏电阻分压电路 → GPIO34（ADC）
   - 继电器控制LED灯带
2. 编写自动灯光程序：
```python
import machine
import time

# 继电器和光敏电阻
relay = machine.Pin(2, machine.Pin.OUT)
light_sensor = machine.ADC(machine.Pin(34))
light_sensor.atten(machine.ADC.ATTN_11DB)

# 阈值
LIGHT_THRESHOLD = 2000

while True:
    light_value = light_sensor.read()
    print("光线强度:", light_value)
    
    if light_value < LIGHT_THRESHOLD:  # 光线暗
        relay.value(1)  # 开灯
        print("开灯")
    else:  # 光线亮
        relay.value(0)  # 关灯
        print("关灯")
    
    time.sleep(1)
```
3. 测试自动灯光功能
4. 遮挡光敏电阻，观察灯光变化

**观察与思考**:
- 光线多暗时开灯合适？
- 如何避免频繁开关？（加入延时或滞后）
- 这个技术可以用在哪些场景？

### 实验2: 🌡️ 温度控制风扇

**零件清单**:
- ESP32开发板一块
- 继电器模块一个
- DHT11温湿度传感器
- 小风扇（5V USB风扇）
- 面包板和杜邦线

**玩法描述**:
1. 连接电路：
   - 继电器信号输入 → GPIO2
   - DHT11数据引脚 → GPIO4
   - 继电器控制风扇
2. 编写温度控制程序：
```python
import machine
import dht
import time

# 继电器和传感器
relay = machine.Pin(2, machine.Pin.OUT)
sensor = dht.DHT11(machine.Pin(4))

# 温度阈值
TEMP_HIGH = 28  # 高于此温度开风扇
TEMP_LOW = 25   # 低于此温度关风扇

fan_state = False

while True:
    try:
        sensor.measure()
        temp = sensor.temperature()
        humi = sensor.humidity()
        
        print("温度:", temp, "湿度:", humi)
        
        # 带滞后的温度控制
        if temp >= TEMP_HIGH and not fan_state:
            relay.value(1)  # 开风扇
            fan_state = True
            print("开风扇")
        elif temp <= TEMP_LOW and fan_state:
            relay.value(0)  # 关风扇
            fan_state = False
            print("关风扇")
        
    except Exception as e:
        print("错误:", e)
    
    time.sleep(2)
```
3. 测试温度控制功能
4. 用手加热传感器，观察风扇变化

**观察与思考**:
- 为什么需要两个阈值（TEMP_HIGH和TEMP_LOW）？
- 如果只有一个阈值会怎样？
- 如何让控制更智能？

### 实验3: 📱 远程控制

**零件清单**:
- ESP32开发板一块
- 继电器模块2个
- DHT11传感器
- 光敏电阻
- LED灯带和风扇
- 电脑

**玩法描述**:
1. 编写Web控制界面：
```python
import network
import socket
import machine
import dht
import time

# WiFi连接
SSID = "你的WiFi名称"
PASSWORD = "你的WiFi密码"

wlan = network.WLAN(network.STA_IF)
wlan.active(True)
wlan.connect(SSID, PASSWORD)

while not wlan.isconnected():
    time.sleep(1)

# 设备初始化
relay1 = machine.Pin(2, machine.Pin.OUT)  # 灯
relay2 = machine.Pin(4, machine.Pin.OUT)  # 风扇
sensor = dht.DHT11(machine.Pin(15))

# HTML页面
def get_html():
    try:
        sensor.measure()
        temp = sensor.temperature()
        humi = sensor.humidity()
    except:
        temp = 0
        humi = 0
    
    light_state = "ON" if relay1.value() else "OFF"
    fan_state = "ON" if relay2.value() else "OFF"
    
    html = f"""<!DOCTYPE html>
<html>
<head><title>Smart Home</title></head>
<body>
<h1>Smart Home Control</h1>
<p>Temperature: {temp}°C</p>
<p>Humidity: {humi}%</p>
<h2>Devices</h2>
<p>Light: {light_state} 
   <a href="/light/on"><button>ON</button></a>
   <a href="/light/off"><button>OFF</button></a>
</p>
<p>Fan: {fan_state}
   <a href="/fan/on"><button>ON</button></a>
   <a href="/fan/off"><button>OFF</button></a>
</p>
<meta http-equiv="refresh" content="5">
</body>
</html>
"""
    return html

# Web服务器
addr = socket.getaddrinfo('0.0.0.0', 80)[0][-1]
s = socket.socket()
s.bind(addr)
s.listen(1)

while True:
    cl, addr = s.accept()
    request = cl.recv(1024)
    request = str(request)
    
    # 解析请求
    if '/light/on' in request:
        relay1.value(1)
    elif '/light/off' in request:
        relay1.value(0)
    elif '/fan/on' in request:
        relay2.value(1)
    elif '/fan/off' in request:
        relay2.value(0)
    
    # 发送响应
    cl.send('HTTP/1.0 200 OK\r\nContent-type: text/html\r\n\r\n')
    cl.send(get_html())
    cl.close()
```
2. 用手机浏览器访问ESP32的IP地址
3. 远程控制灯光和风扇
4. 查看实时温湿度数据

**观察与思考**:
- 远程控制有什么优势？
- 如何保证安全性？（密码、加密）
- 还可以控制哪些设备？

## 🧠 核心原理简述（给家长）

**避障算法**是自主机器人的基础能力。它结合了传感器数据、决策逻辑和运动控制。

核心概念：
- **感知**: 用超声波传感器检测障碍物距离
- **决策**: 根据距离决定停车、转向或绕行
- **执行**: 控制电机完成决策动作
- **循环**: 不断重复感知-决策-执行

常见避障策略：
1. **遇障停车**: 检测到障碍物就停车
2. **遇障转向**: 检测到障碍物就转向
3. **随机漫步**: 随机选择方向，避免卡住
4. **沿墙行走**: 沿着障碍物边缘行走

## 🔬 趣味实验清单

### 实验1: 🛑 遇障停车

**零件清单**:
- Maqueen小车套件
- Micro:bit V2 一块
- 超声波传感器（HC-SR04）
- 杜邦线4根
- 电池
- 障碍物（纸箱、书本）

**玩法描述**:
1. 安装超声波传感器到Maqueen
2. 连接电路：
   - VCC → 3V
   - GND → GND
   - Trig → P1
   - Echo → P2
3. 编写避障程序：
```python
from microbit import *
import maqueen

def get_distance():
    pin1.write_digital(1)
    sleep(0.01)
    pin1.write_digital(0)
    duration = pin2.read_pulse(1)
    distance = duration * 0.034 / 2
    return distance

while True:
    dist = get_distance()
    
    if dist < 20:  # 障碍物太近
        maqueen.motor_stop(maqueen.Motors.ALL)
        display.scroll("STOP!")
    else:
        maqueen.motor_run(maqueen.Motors.ALL, maqueen.Dir.CW, 80)
    
    sleep(100)
```
4. 测试避障功能

**观察与思考**:
- 多远距离开始停车？
- 停车反应快吗？有延迟吗？
- 如何调整停车距离？

### 实验2: 🔄 自动转向

**零件清单**:
- 同实验1

**玩法描述**:
1. 编写转向避障程序：
```python
from microbit import *
import maqueen
import random

def get_distance():
    pin1.write_digital(1)
    sleep(0.01)
    pin1.write_digital(0)
    duration = pin2.read_pulse(1)
    distance = duration * 0.034 / 2
    return distance

while True:
    dist = get_distance()
    
    if dist < 20:  # 前方有障碍
        # 后退一点
        maqueen.motor_run(maqueen.Motors.ALL, maqueen.Dir.CCW, 60)
        sleep(500)
        
        # 随机选择左转或右转
        if random.randint(0, 1) == 0:
            maqueen.motor_run(maqueen.Motors.M1, maqueen.Dir.CCW, 80)
            maqueen.motor_run(maqueen.Motors.M2, maqueen.Dir.CW, 80)
        else:
            maqueen.motor_run(maqueen.Motors.M1, maqueen.Dir.CW, 80)
            maqueen.motor_run(maqueen.Motors.M2, maqueen.Dir.CCW, 80)
        sleep(800)
    else:
        # 前进
        maqueen.motor_run(maqueen.Motors.ALL, maqueen.Dir.CW, 80)
    
    sleep(100)
```
2. 在复杂环境中测试
3. 观察小车能否避开多个障碍物

**观察与思考**:
- 随机转向和固定转向，哪个更好？
- 小车会不会卡在角落里？
- 如何让转向更"智能"？

### 实验3: 🎲 随机漫步

**零件清单**:
- 同实验1

**玩法描述**:
1. 编写随机漫步程序：
```python
from microbit import *
import maqueen
import random

def get_distance():
    pin1.write_digital(1)
    sleep(0.01)
    pin1.write_digital(0)
    duration = pin2.read_pulse(1)
    distance = duration * 0.034 / 2
    return distance

def random_turn():
    direction = random.choice(["left", "right"])
    duration = random.randint(300, 1000)
    
    if direction == "left":
        maqueen.motor_run(maqueen.Motors.M1, maqueen.Dir.CCW, 80)
        maqueen.motor_run(maqueen.Motors.M2, maqueen.Dir.CW, 80)
    else:
        maqueen.motor_run(maqueen.Motors.M1, maqueen.Dir.CW, 80)
        maqueen.motor_run(maqueen.Motors.M2, maqueen.Dir.CCW, 80)
    
    sleep(duration)

while True:
    dist = get_distance()
    
    if dist < 25:  # 遇到障碍
        # 后退
        maqueen.motor_run(maqueen.Motors.ALL, maqueen.Dir.CCW, 60)
        sleep(500)
        # 随机转向
        random_turn()
    else:
        # 随机前进或转向
        action = random.choice(["forward", "turn"])
        if action == "forward":
            maqueen.motor_run(maqueen.Motors.ALL, maqueen.Dir.CW, 80)
            sleep(random.randint(500, 2000))
        else:
            random_turn()
    
    sleep(100)
```
2. 在封闭空间中测试
3. 观察小车的移动路径

**观察与思考**:
- 小车能探索整个空间吗？
- 有没有覆盖不到的区域？
- 随机漫步有什么优缺点？
